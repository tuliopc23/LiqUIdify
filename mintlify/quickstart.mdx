## Installation

Choose your preferred package manager to install LiqUIdify:

<CodeGroup>

```bash npm
npm install @liquidify/components
```

```bash yarn
yarn add @liquidify/components
```

```bash pnpm
pnpm add @liquidify/components
```

```bash bun
bun add @liquidify/components
```

</CodeGroup>

## Setup

### 1. Import Styles

Add the LiqUIdify styles to your application. You can import them in your main CSS file or JavaScript entry point:

<CodeGroup>

```css CSS Import
/* In your global CSS file */
@import '@liquidify/components/css';
```

```jsx JavaScript Import
// In your main JavaScript/TypeScript file
import '@liquidify/components/css';
```

```jsx Next.js
// In your _app.js or _app.tsx
import '@liquidify/components/css';

function MyApp({ Component, pageProps }) {
  return <Component {...pageProps} />;
}
```

</CodeGroup>

### 2. Start Using Components

Now you can start using LiqUIdify components in your application:

```jsx
import { GlassButton, GlassCard, GlassInput } from '@liquidify/components';

function App() {
  return (
    <div className="min-h-screen bg-gradient-to-br from-purple-400 to-blue-600 p-8">
      <GlassCard className="max-w-md mx-auto">
        <h1 className="text-2xl font-bold mb-4">Welcome to LiqUIdify</h1>
        <GlassInput
          placeholder="Enter your email"
          className="mb-4"
        />
        <GlassButton variant="primary" fullWidth>
          Get Started
        </GlassButton>
      </GlassCard>
    </div>
  );
}
```

## TypeScript Setup

LiqUIdify is built with TypeScript and includes comprehensive type definitions. No additional setup is required - types are automatically included when you install the package.

```tsx
import { GlassButton, GlassButtonProps } from '@liquidify/components';

// Type-safe props
const MyButton: React.FC<GlassButtonProps> = (props) => {
  return <GlassButton {...props} />;
};
```

## Tree-Shaking Support

LiqUIdify supports tree-shaking out of the box. You can import components individually to reduce bundle size:

```jsx
// ✅ Recommended: Import only what you need
import { GlassButton } from '@liquidify/components/button';
import { GlassCard } from '@liquidify/components/card';

// ❌ Avoid: Importing everything
import * as Liquid from '@liquidify/components';
```
