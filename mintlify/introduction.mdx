## Welcome to LiqUIdify

LiqUIdify is a cutting-edge React component library that brings the elegance of glassmorphism design to your applications. With over 50+ beautifully crafted components, LiqUIdify provides everything you need to build modern, visually stunning user interfaces.

## Key Features

- **50+ Components**: From basic buttons to complex data tables, we've got you covered
- **Glassmorphism Effects**: Beautiful frosted glass aesthetics with customizable blur and transparency
- **TypeScript First**: Full type safety with comprehensive TypeScript definitions
- **Accessibility**: WCAG 2.1 compliant with keyboard navigation and screen reader support
- **Dark Mode**: Built-in dark mode support with automatic theme detection
- **Customizable**: Extensive theming options via CSS variables and component props
- **Tree-Shakeable**: Import only what you need for optimal bundle size
- **SSR Compatible**: Works seamlessly with Next.js, Gatsby, and other SSR frameworks

## Quick Example

```jsx
import { GlassButton, GlassCard } from '@liquidify/components';

function App() {
  return (
    <GlassCard blur="medium" opacity={0.8}>
      <h2>Welcome to LiqUIdify</h2>
      <p>Build beautiful glassmorphism UIs with ease</p>
      <GlassButton variant="primary" glow>
        Get Started
      </GlassButton>
    </GlassCard>
  );
}
```
