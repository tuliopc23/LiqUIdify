## Basic Usage

<CodeGroup>

```tsx React Component
import {
  GlassButton,
  GlassCard,
  UnifiedGlassProvider,
} from "@liquidify/components";
import "@liquidify/components/css";

function App() {
  return (
    <UnifiedGlassProvider>
      <GlassCard variant="elevated">
        <h1>Welcome to LiqUIdify</h1>
        <GlassButton variant="primary" size="large">
          Get Started
        </GlassButton>
      </GlassCard>
    </UnifiedGlassProvider>
  );
}
```

```tsx Next.js
// pages/_app.tsx
import { UnifiedGlassProvider } from "@liquidify/components";
import "@liquidify/components/css";

export default function App({ Component, pageProps }) {
  return (
    <UnifiedGlassProvider>
      <Component {...pageProps} />
    </UnifiedGlassProvider>
  );
}
```

</CodeGroup>

<Warning>
  Make sure to wrap your app with `UnifiedGlassProvider` for optimal glass
  effects and theme management.
</Warning>

## What's Included

<Accordion title="Core Components">
  - **GlassButton**: Primary, secondary, ghost, and destructive variants -
  **GlassCard**: Elevated, flat, and interactive cards with compound components
  - **GlassInput**: Text inputs with validation and error states -
  **GlassModal**: Accessible modals with focus trap and backdrop
</Accordion>

<Accordion title="Form Components">
  - **GlassCheckbox**: Single and group checkboxes with indeterminate states -
  **GlassRadioGroup**: Radio button groups with custom styling -
  **GlassSelect**: Dropdown selects with search and multi-select -
  **GlassTextarea**: Resizable text areas with character counting -
  **GlassFileUpload**: Drag-and-drop file uploads with progress
</Accordion>

<Accordion title="Layout & Navigation">
  - **GlassAccordion**: Collapsible content sections - **GlassBreadcrumbs**:
  Navigation breadcrumbs with separators - **GlassTabs**: Horizontal and
  vertical tab navigation - **GlassNavbar**: Responsive navigation with mobile
  support - **GlassSidebar**: Collapsible sidebar navigation
</Accordion>

<Accordion title="Feedback & Data">
  - **GlassAlert**: Success, warning, error, and info alerts - **GlassToast**:
  Non-blocking notifications with auto-dismiss - **GlassProgress**: Linear and
  circular progress indicators - **GlassTable**: Sortable, filterable data
  tables - **GlassSkeleton**: Loading placeholders with animation
</Accordion>
