# GlassButton

The `GlassButton` component provides a beautiful glassmorphism button with various styles, sizes, and interactive states.

## Import

```tsx
import { GlassButton } from "@liquidify/components";
```

## Basic Usage

## Sizes

The button supports five different sizes:

```tsx
<GlassButton size="xs">Extra Small</GlassButton>
<GlassButton size="sm">Small</GlassButton>
<GlassButton size="md">Medium</GlassButton>
<GlassButton size="lg">Large</GlassButton>
<GlassButton size="xl">Extra Large</GlassButton>
```

## Glass Effects

Control the glass effect intensity:

```tsx
<GlassButton glassEffect={{ intensity: "low" }}>
  Low Intensity
</GlassButton>

<GlassButton glassEffect={{ intensity: "medium" }}>
  Medium Intensity
</GlassButton>

<GlassButton glassEffect={{ intensity: "high" }}>
  High Intensity
</GlassButton>

<GlassButton glassEffect={{ intensity: "ultra" }}>
  Ultra Intensity
</GlassButton>
```

## API Reference

<ResponseField name="variant" type="string" default="primary">
  The visual style variant of the button Options: `primary` | `secondary` |
  `ghost` | `destructive` | `tertiary` | `apple`
</ResponseField>

<ResponseField name="size" type="string" default="md">
  The size of the button Options: `xs` | `sm` | `md` | `lg` | `xl`
</ResponseField>

<ResponseField name="disabled" type="boolean" default="false">
  Whether the button is disabled
</ResponseField>

<ResponseField name="loading" type="boolean" default="false">
  Whether the button is in a loading state
</ResponseField>

<ResponseField name="asChild" type="boolean" default="false">
  Render as a child component (useful for Next.js Link)
</ResponseField>

<ResponseField name="glassEffect" type="GlassEffect">
  Glass effect configuration
  
  ```tsx
  interface GlassEffect {
    intensity?: "low" | "medium" | "high" | "ultra"
    blur?: boolean
    backdrop?: boolean
  }
  ```
</ResponseField>

<ResponseField name="animationTiming" type="string" default="normal">
  Animation timing for state transitions Options: `instant` | `fast` | `normal`
  | `slow` | `slower`
</ResponseField>

<ResponseField name="children" type="ReactNode">
  The content of the button
</ResponseField>

<ResponseField name="onClick" type="function">
  Click event handler
  
  ```tsx
  (event: React.MouseEvent<HTMLButtonElement>) => void
  ```
</ResponseField>

## Accessibility

The `GlassButton` component follows WCAG 2.1 AA guidelines:

<AccordionGroup>

<Accordion title="Keyboard Navigation">
  - **Tab**: Navigate to the button - **Enter/Space**: Activate the button -
  **Escape**: Blur focus (when focused)
</Accordion>

<Accordion title="Screen Readers">
  - Proper ARIA attributes for button role - Loading state announced to screen
  readers - Disabled state properly communicated - Support for `aria-label` and
  `aria-describedby`
</Accordion>

<Accordion title="Focus Management">
  - Visible focus indicators - Focus trap support when used in modals - Proper
  tab order
</Accordion>

</AccordionGroup>
