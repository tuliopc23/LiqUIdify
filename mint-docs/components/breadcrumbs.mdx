# GlassBreadcrumbs

The `GlassBreadcrumbs` component provides a navigation trail that shows the user's location within a website hierarchy with glassmorphism styling.

## Import

```tsx
import { GlassBreadcrumbs } from "@liquidify/components";
```

## Basic Usage

<CodeGroup>

```tsx Basic Breadcrumbs
<GlassBreadcrumbs>
  <GlassBreadcrumbs.Item href="/">Home</GlassBreadcrumbs.Item>
  <GlassBreadcrumbs.Separator />
  <GlassBreadcrumbs.Item href="/docs">Documentation</GlassBreadcrumbs.Item>
  <GlassBreadcrumbs.Separator />
  <GlassBreadcrumbs.Item current>Components</GlassBreadcrumbs.Item>
</GlassBreadcrumbs>
```

## Component API

### GlassBreadcrumbs

| Prop        | Type                     | Default     | Description                             |
| ----------- | ------------------------ | ----------- | --------------------------------------- |
| `variant`   | `'default' \| 'minimal'` | `'default'` | Visual variant                          |
| `size`      | `'sm' \| 'md' \| 'lg'`   | `'md'`      | Breadcrumb size                         |
| `maxItems`  | `number`                 | -           | Maximum items to show before collapsing |
| `separator` | `ReactNode`              | `'/'`       | Default separator                       |

### GlassBreadcrumbs.Item

| Prop       | Type         | Default | Description            |
| ---------- | ------------ | ------- | ---------------------- |
| `href`     | `string`     | -       | Link URL               |
| `onClick`  | `() => void` | -       | Click handler          |
| `current`  | `boolean`    | `false` | Current page indicator |
| `icon`     | `ReactNode`  | -       | Item icon              |
| `disabled` | `boolean`    | `false` | Disable item           |

### GlassBreadcrumbs.Separator

| Prop   | Type        | Default | Description           |
| ------ | ----------- | ------- | --------------------- |
| `icon` | `ReactNode` | -       | Custom separator icon |

## Examples

### Dynamic Breadcrumbs

```tsx
function DynamicBreadcrumbs({ path }: { path: string }) {
  const pathSegments = path.split("/").filter(Boolean);

  return (
    <GlassBreadcrumbs>
      <GlassBreadcrumbs.Item href="/" icon={<HomeIcon />}>
        Home
      </GlassBreadcrumbs.Item>

      {pathSegments.map((segment, index) => {
        const href = "/" + pathSegments.slice(0, index + 1).join("/");
        const isLast = index === pathSegments.length - 1;

        return (
          <React.Fragment key={segment}>
            <GlassBreadcrumbs.Separator />
            <GlassBreadcrumbs.Item
              href={isLast ? undefined : href}
              current={isLast}
            >
              {segment.charAt(0).toUpperCase() + segment.slice(1)}
            </GlassBreadcrumbs.Item>
          </React.Fragment>
        );
      })}
    </GlassBreadcrumbs>
  );
}
```


## Accessibility

The `GlassBreadcrumbs` component follows WAI-ARIA guidelines:

- **Navigation Landmark**: Uses `role="navigation"` with `aria-label="Breadcrumb"`
- **Current Page**: Uses `aria-current="page"` for the current item
- **Link Semantics**: Proper link and text semantics for navigation
- **Keyboard Navigation**: Standard keyboard navigation support
