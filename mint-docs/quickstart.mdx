# Quick Start Guide

Get started with LiqUIdify in just a few steps. This guide will help you install the library and create your first glassmorphism component.

<Note>
  Before you begin, make sure you have Node.js 18+ and a React 18+ project set up.
</Note>

## Step 1: Installation

Choose your preferred package manager to install LiqUIdify:

<CodeGroup>

```bash npm
npm install @liquidify/components
```

```bash yarn  
yarn add @liquidify/components
```

```bash pnpm
pnpm add @liquidify/components
```

```bash bun
bun add @liquidify/components
```

</CodeGroup>

## Step 2: Import Styles

Import the LiqUIdify styles in your app's entry point (usually `src/main.tsx`, `src/index.tsx`, or `pages/_app.tsx`):

<CodeGroup>

```tsx React/Vite
// src/main.tsx or src/index.tsx
import "@liquidify/components/css";
import { StrictMode } from "react";
import { createRoot } from "react-dom/client";
import App from "./App.tsx";

createRoot(document.getElementById("root")!).render(
  <StrictMode>
    <App />
  </StrictMode>
);
```

```tsx Next.js
// pages/_app.tsx
import "@liquidify/components/css";
import type { AppProps } from "next/app";

export default function App({ Component, pageProps }: AppProps) {
  return <Component {...pageProps} />;
}
```

```tsx Next.js 13+ (App Router)
// app/layout.tsx
import "@liquidify/components/css";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>{children}</body>
    </html>
  );
}
```

</CodeGroup>

## Step 3: Add the Provider

Wrap your app with the `UnifiedGlassProvider` to enable glass effects and theme management:

<CodeGroup>

```tsx React
// src/App.tsx
import { UnifiedGlassProvider } from "@liquidify/components";
import "@liquidify/components/css";

function App() {
  return (
    <UnifiedGlassProvider>
      <div className="min-h-screen bg-gradient-to-br from-blue-50 to-purple-50">
        {/* Your app content */}
      </div>
    </UnifiedGlassProvider>
  );
}

export default App;
```

```tsx Next.js
// pages/_app.tsx
import { UnifiedGlassProvider } from "@liquidify/components";
import "@liquidify/components/css";
import type { AppProps } from "next/app";

export default function App({ Component, pageProps }: AppProps) {
  return (
    <UnifiedGlassProvider>
      <Component {...pageProps} />
    </UnifiedGlassProvider>
  );
}
```

```tsx Next.js 13+ (App Router)
// app/providers.tsx
"use client";

import { UnifiedGlassProvider } from "@liquidify/components";

export default function Providers({ children }: { children: React.ReactNode }) {
  return <UnifiedGlassProvider>{children}</UnifiedGlassProvider>;
}

// app/layout.tsx
import Providers from "./providers";
import "@liquidify/components/css";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        <Providers>{children}</Providers>
      </body>
    </html>
  );
}
```

</CodeGroup>

## Step 4: Create Your First Component

Now let's create a simple example using LiqUIdify components:

<CodeGroup>

```tsx Basic Example
import { 
  GlassButton, 
  GlassCard, 
  GlassInput 
} from "@liquidify/components";

export default function Welcome() {
  return (
    <div className="min-h-screen bg-gradient-to-br from-blue-50 to-purple-50 p-8">
      <div className="max-w-md mx-auto mt-16">
        <GlassCard variant="elevated" className="p-8">
          <h1 className="text-2xl font-bold text-gray-900 mb-6">
            Welcome to LiqUIdify
          </h1>
          
          <div className="space-y-4">
            <GlassInput 
              placeholder="Enter your email" 
              type="email"
            />
            
            <GlassButton 
              variant="primary" 
              size="lg" 
              fullWidth
            >
              Get Started
            </GlassButton>
          </div>
        </GlassCard>
      </div>
    </div>
  );
}
```

## Core Bundle (Recommended)

For optimal performance, import only the components you need from the core bundle:

```tsx
// Core bundle includes: Button, Card, Input, Modal
import { core } from "@liquidify/components/bundles";
import "@liquidify/components/css";

const { GlassButton, GlassCard, GlassInput, GlassModal } = core;

export default function App() {
  return (
    <UnifiedGlassProvider>
      <GlassCard>
        <GlassInput placeholder="Enter your name" />
        <GlassButton>Submit</GlassButton>
      </GlassCard>
    </UnifiedGlassProvider>
  );
}
```

## TypeScript Support

LiqUIdify is built with TypeScript and provides excellent IntelliSense support:

```tsx
import { GlassButton, type GlassButtonProps } from "@liquidify/components";

// Full type safety with IntelliSense
const ButtonExample: React.FC<GlassButtonProps> = (props) => {
  return (
    <GlassButton
      variant="primary" // ✅ Autocomplete: primary | secondary | ghost | destructive
      size="lg" // ✅ Autocomplete: xs | sm | md | lg | xl
      disabled={false} // ✅ Type-checked boolean
      {...props}
    >
      Click me
    </GlassButton>
  );
};
```
