# Hooks API

LiqUIdify provides a comprehensive set of custom React hooks for building glassmorphism interfaces, managing state, and enhancing user experience.

## Glass Effect Hooks

### useGlassEffect

Core hook for managing glassmorphism visual effects.

<Card title="Import">
  ```tsx import {useGlassEffect} from "@liquidify/components/hooks"; ```
</Card>

#### Usage

```tsx
const glassProps = useGlassEffect({
  intensity: "medium",
  variant: "primary",
  interactive: true,
  animated: true,
});

return <div {...glassProps}>Glass Element</div>;
```

#### Parameters

| Parameter     | Type                                     | Default     | Description          |
| ------------- | ---------------------------------------- | ----------- | -------------------- |
| `intensity`   | `'low' \| 'medium' \| 'high' \| 'ultra'` | `'medium'`  | Glass blur intensity |
| `variant`     | `'default' \| 'primary' \| 'secondary'`  | `'default'` | Glass color variant  |
| `interactive` | `boolean`                                | `false`     | Enable hover effects |
| `animated`    | `boolean`                                | `true`      | Enable transitions   |

#### Returns

```tsx
{
  className: string;
  style: CSSProperties;
  'data-glass-intensity': string;
  'data-glass-variant': string;
}
```

### useGlassAnimation

Advanced hook for glass animation effects and transitions.

<Card title="Import">
  ```tsx import {useGlassAnimation} from "@liquidify/components/hooks"; ```
</Card>

#### Usage

```tsx
const { animate, isAnimating, controls } = useGlassAnimation({
  type: "shimmer",
  duration: 1000,
  autoPlay: true,
});

// Trigger animation
const handleClick = () => {
  animate("pulse");
};
```

#### Parameters

| Parameter  | Type                                                   | Default      | Description             |
| ---------- | ------------------------------------------------------ | ------------ | ----------------------- |
| `type`     | `'fade' \| 'slide' \| 'scale' \| 'shimmer' \| 'pulse'` | `'fade'`     | Animation type          |
| `duration` | `number`                                               | `300`        | Animation duration (ms) |
| `easing`   | `string`                                               | `'ease-out'` | CSS easing function     |
| `autoPlay` | `boolean`                                              | `false`      | Auto-start animation    |

### useGlassTheme

Hook for managing glass theme and color schemes.

<Card title="Import">
  ```tsx import {useGlassTheme} from "@liquidify/components/hooks"; ```
</Card>

#### Usage

```tsx
const { theme, setTheme, isDark, colors, toggleTheme } = useGlassTheme();

// Change theme
setTheme("dark");

// Get theme colors
const primaryColor = colors.primary;
```

#### Returns

```tsx
{
  theme: 'light' | 'dark' | 'auto';
  setTheme: (theme: Theme) => void;
  isDark: boolean;
  colors: ThemeColors;
  toggleTheme: () => void;
}
```

## State Management Hooks

### useGlassState

Universal state management hook with glass-specific optimizations.

<Card title="Import">
  ```tsx import {useGlassState} from "@liquidify/components/hooks"; ```
</Card>

#### Usage

```tsx
const [state, setState, actions] = useGlassState({
  isOpen: false,
  isLoading: false,
  data: null,
});

// Update state
actions.toggle("isOpen");
actions.set("isLoading", true);
actions.reset();
```

#### Parameters

| Parameter      | Type                  | Description           |
| -------------- | --------------------- | --------------------- |
| `initialState` | `Record<string, any>` | Initial state object  |
| `options`      | `StateOptions`        | Configuration options |

#### Usage

```tsx
const id = useGlassId("button");
```

| Parameter | Type     | Default   | Description |
| --------- | -------- | --------- | ----------- |

### useGlassDebounce

Debounce values and callbacks.

<Card title="Import">
  ```tsx import {useGlassDebounce} from "@liquidify/components/hooks"; ```
</Card>

#### Usage

```tsx

    performSearch(debouncedSearchTerm);
  }
}, [debouncedSearchTerm]);
```

## Form Hooks

### useGlassForm

Comprehensive form state management.

<Card title="Import">
  ```tsx import {useGlassForm} from "@liquidify/components/hooks"; ```
</Card>

#### Usage

```tsx
const form = useGlassForm({
```

## Hook Composition

Many hooks can be composed together for complex functionality:

```tsx
function useAdvancedGlassComponent() {
  const theme = useGlassTheme();
  const glassEffect = useGlassEffect({
    intensity: theme.isDark ? "high" : "medium",
    variant: "primary",
  });
  const animation = useGlassAnimation({ type: "shimmer" });
  const [state, setState] = useGlassState({ isActive: false });

  return {
    ...glassEffect,
    ...animation,
    state,
    setState,
    theme,
  };
}
```

## TypeScript Support

