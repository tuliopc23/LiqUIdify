# Getting Started
## Quick path
Fastest way to try LiqUIdify in your project.

### Prerequisites
- Node and a package manager (bun, npm, pnpm, or yarn)
- React 18 or 19

### Steps
1. Install the package and peers (see Installing).
2. Import the styles once in your app entry.
3. Import components (root or subpath).
