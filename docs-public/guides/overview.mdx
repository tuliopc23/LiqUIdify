# Overview
## What is LiqUIdify?
LiqUIdify is a React component library with Apple HIG-inspired design, built on Ark UI primitives and Panda CSS tokens. It focuses on clarity, motion, and a polished liquid glass aesthetic.

### When to use it
Use LiqUIdify for production-grade UI with accessible interactions, strong defaults, and ergonomic imports.

### Core concepts
- Components wrap Ark UI for accessible behavior
- Styling uses Panda tokens and generated CSS
- Subpath imports enable fine-grained tree-shaking
