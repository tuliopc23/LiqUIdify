# Installing
## Install the package and peers
Always use the latest stable versions.

### Commands (fish shell)
Use your preferred package manager.

```bash
# bun
bun add liquidify-react@latest react@latest react-dom@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest

# npm
npm i liquidify-react@latest react@latest react-dom@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest

# pnpm
pnpm add liquidify-react@latest react@latest react-dom@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest

# yarn
yarn add liquidify-react@latest react@latest react-dom@latest @ark-ui/react@latest framer-motion@latest lucide-react@latest
```

### Notes
- The CSS export is provided at liquidify-react/styles
- React and React DOM are peers and must already be in your app
