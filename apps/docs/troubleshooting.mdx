description: Fix common issues fast.

## TypeScript (strict)

- Ensure React types match your runtime: `react` and `@types/react` v18/19.
- If generics fail, upgrade TS `^5.9`.

## CSS not applying

- Import once: `import 'liquidify/styles'`.
- Confirm Tailwind content globs include node_modules distribution.

## ESM/CJS mismatch

- Use ESM bundlers (Vite/Rolldown). Avoid `require()` imports.

## React version

- Peer deps: `react`/`react-dom` `^18 || ^19`.

## Vite config

- Keep `optimizeDeps` default. Do not alias to `src/`.
