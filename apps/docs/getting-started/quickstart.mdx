# Quick Start Guide

Get up and running with LiqUIdify’s liquid glass UI in minutes. Install, wrap your app with the provider, and drop in a component.

## 📦 Installation

```bash
bun add liquidify
```

## 🎨 Setup Provider

Wrap your app with `ThemeProvider` and import the CSS once:

```tsx
import "liquidify/styles";
import { ThemeProvider } from "liquidify";

function App() {
  return <ThemeProvider>{/* your app */}</ThemeProvider>;
}
```

## 🌊 Your First Liquid Glass Component

```tsx
import { Card, Button } from "liquidify";

export default function WelcomeCard() {
  return (
    <GlassCard variant="elevated" className="p-8 max-w-md mx-auto">
      <GlassCard.Header>
        <GlassCard.Title>Welcome to LiqUIdify</GlassCard.Title>
        <GlassCard.Description>
          Experience beautiful liquid glass morphism with Apple-inspired design.
        </GlassCard.Description>
      </GlassCard.Header>

      <GlassCard.Content>
        <p className="text-white/80 mb-6">
          Build stunning interfaces with smooth animations and visual depth.
        </p>
      </GlassCard.Content>

      <GlassCard.Actions>
        <GlassButton variant="primary" size="lg">
          Get Started
        </GlassButton>
        <GlassButton variant="ghost">Learn More</GlassButton>
      </GlassCard.Actions>
    </Card>
  );
}
```


## 🌙 Theme Toggle

```tsx
import { useTheme, Button } from "liquidify";

function ThemeToggle() {
  const { theme, setTheme } = useTheme();
  return (
    <GlassButton
      onClick={() => setTheme(theme === "light" ? "dark" : "light")}
      variant="ghost"
    >
      {theme === "light" ? "🌙" : "☀️"} Toggle Theme
    </GlassButton>
  );
}
```
