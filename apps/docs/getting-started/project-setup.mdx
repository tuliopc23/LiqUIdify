This page shows the minimal setup required to start using the library. Works with Vite, Bun, and React 18/19.

## 1) Install

```bash
bun add liquidify
```

## 2) Import styles

Import the library CSS once (tree-shake safe). Add to `src/main.tsx` or your app root:

```tsx
import "liquidify/styles";
```

That's it. No additional CSS framework is required.

## 3) Optional: PostCSS (autoprefixer)

If your app uses PostCSS, enable autoprefixer:

```js
// postcss.config.js or postcss.config.ts
export default { plugins: { autoprefixer: {} } };
```

## 4) TypeScript config (optional)

To improve DX with ESM and modern resolution:

```json
{
  "compilerOptions": {
    "moduleResolution": "bundler",
    "allowSyntheticDefaultImports": true
  }
}
```

## 5) ESM + sideEffects

The package ships ESM with sideEffects set for CSS. Keep `build.target` modern, and avoid importing from `src/`.
