# Framework Integration

LiqUIdify works seamlessly with all modern React frameworks. This guide provides specific instructions for integrating the library with popular frameworks, handling SSR, and optimizing performance.

## üöÄ Next.js Integration

### Installation & Setup

1. **Install dependencies**:

   ```bash
   npm install liquidify
   # or
   bun add liquidify
   ```
2. **Configure Next.js app (App Router)**:

   ```tsx
   // app/layout.tsx
   import "liquidify/styles";
   
   export default function RootLayout({
     children,
   }: {
     children: React.ReactNode;
   }) {
     return (
       <html lang="en">
         <body>
           <UnifiedGlassProvider theme="light">{children}</UnifiedGlassProvider>
         </body>
       </html>
     );
   }
   ```
3. **Pages Router setup**:

   ```tsx
   // pages/_app.tsx
   import type { AppProps } from "next/app";
   import "liquidify/styles";
   
   export default function App({ Component, pageProps }: AppProps) {
     return (
       <UnifiedGlassProvider theme="light">
         <Component {...pageProps} />
       </UnifiedGlassProvider>
     );
   }
   ```

### SSR Considerations

LiqUIdify handles SSR automatically, but for optimal performance:

1. **Use dynamic imports for heavy components**:

   ```tsx
   import dynamic from "next/dynamic";
   
   const GlassChart = dynamic(
     () => import("liquidify").then((mod) => mod.GlassChart),
     {
       ssr: false,
       loading: () => <div>Loading chart...</div>,
     },
   );
   ```
2. **Handle theme hydration**:

   ```tsx
   // app/providers.tsx
   "use client";
   
   export function Providers({ children }: { children: React.ReactNode }) {
     const [mounted, setMounted] = useState(false);
   
     useEffect(() => {
       setMounted(true);
     }, []);
   
     return (
       <UnifiedGlassProvider
         theme={mounted ? "light" : undefined}
         suppressHydrationWarning
       >
         {children}
       </UnifiedGlassProvider>
     );
   }
   ```

### Performance Optimization

1. **Configure next.config.js for optimal bundling**:

   ```js
   // next.config.js
   module.exports = {
     transpilePackages: ["liquidify"],
     experimental: {
       optimizeCss: true,
     },
     compiler: {
       removeConsole: process.env.NODE_ENV === "production",
     },
   };
   ```
2. **Use Next.js Image with Glass components**:

   ```tsx
   import Image from "next/image";
   export function ProductCard() {
     return (
       <GlassCard variant="elevated">
         <Image
           src="/product.jpg"
           alt="Product"
           width={300}
           height={200}
           className="rounded-lg"
         />
         <GlassCard.Title>Product Name</GlassCard.Title>
       </GlassCard>
     );
   }
   ```

### Example: Complete Next.js Setup

```tsx
// app/layout.tsx
import { Inter } from "next/font/google";
import { Providers } from "./providers";
import "liquidify/styles";
import "./globals.css";

const inter = Inter({ subsets: ["latin"] });

export const metadata = {
  title: "My App with LiqUIdify",
  description: "Beautiful glassmorphism UI",
};

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body className={inter.className}>
        <Providers>{children}</Providers>
      </body>
    </html>
  );
}
```

## ‚ö° Vite Integration

### Installation & Setup

1. **Create Vite project with React**:

   ```bash
   npm create vite@latest my-app -- --template react-ts
   cd my-app
   npm install liquidify
   ```
2. **Configure main entry**:

   ```tsx
   // src/main.tsx
   import ReactDOM from "react-dom/client";
   import App from "./App";
   import "liquidify/styles";
   import "./index.css";
   
   ReactDOM.createRoot(document.getElementById("root")!).render(
     <React.StrictMode>
       <UnifiedGlassProvider theme="light">
         <App />
       </UnifiedGlassProvider>
     </React.StrictMode>,
   );
   ```
3. **Optimize Vite config**:

   ```ts
   // vite.config.ts
   import { defineConfig } from "vite";
   export default defineConfig({
     plugins: [react()],
     optimizeDeps: {
       include: ["liquidify"],
     },
     build: {
       rollupOptions: {
         output: {
           manualChunks: {
             liquidify: ["liquidify"],
           },
         },
       },
     },
   });
   ```

### Development Optimizations

1. **Fast Refresh configuration**:

   ```ts
   // vite.config.ts
   export default defineConfig({
     plugins: [
       react({
         fastRefresh: true,
         include: "**/*.tsx",
       }),
     ],
   });
   ```
2. **CSS optimization with PostCSS**:

   ```js
   // postcss.config.js
   export default {
     plugins: {
       tailwindcss: {},
       autoprefixer: {},
       ...(process.env.NODE_ENV === "production" ? { cssnano: {} } : {}),
     },
   };
   ```

## üé≠ Remix Integration

### Installation & Setup

1. **Install in Remix project**:

   ```bash
   npm install liquidify
   ```
2. **Configure root route**:

   ```tsx
   // app/root.tsx
   import {
     Links,
     LiveReload,
     Meta,
     Outlet,
     Scripts,
     ScrollRestoration,
   } from "@remix-run/react";
   import liquidifyStyles from "liquidify/styles";
   
   export const links: LinksFunction = () => [
     { rel: "stylesheet", href: liquidifyStyles },
   ];
   
   export default function App() {
     return (
       <html lang="en">
         <head>
           <Meta />
           <Links />
         </head>
         <body>
           <UnifiedGlassProvider theme="light">
             <Outlet />
           </UnifiedGlassProvider>
           <ScrollRestoration />
           <Scripts />
           <LiveReload />
         </body>
       </html>
     );
   }
   ```

### SSR & Hydration

1. **Handle client-only components**:

   ```tsx
   // app/components/ClientOnly.tsx
   import { useHydrated } from "~/hooks/use-hydrated";
   
   export function ClientOnly({
     children,
     fallback = null,
   }: {
     children: React.ReactNode;
     fallback?: React.ReactNode;
   }) {
     const hydrated = useHydrated();
     return hydrated ? <>{children}</> : <>{fallback}</>;
   }
   
   // Usage
   export default function Dashboard() {
     return (
       <ClientOnly fallback={<div>Loading chart...</div>}>
         <GlassChart data={data} />
       </ClientOnly>
     );
   }
   ```
2. **Streaming SSR support**:

   ```tsx
   // app/entry.server.tsx
   import { PassThrough } from "stream";
   import { renderToPipeableStream } from "react-dom/server";
   export default function handleRequest(
     request: Request,
     responseStatusCode: number,
     responseHeaders: Headers,
     remixContext: EntryContext,
   ) {
     return new Promise((resolve, reject) => {
       const { pipe, abort } = renderToPipeableStream(
         <RemixServer context={remixContext} url={request.url} />,
         {
           onShellReady() {
             const body = new PassThrough();
             responseHeaders.set("Content-Type", "text/html");
   
             resolve(
               new Response(body, {
                 headers: responseHeaders,
                 status: responseStatusCode,
               }),
             );
   
             pipe(body);
           },
           onShellError(error) {
             reject(error);
           },
         },
       );
   
       setTimeout(abort, 5000);
     });
   }
   ```

## üîß Common Integration Patterns

### TypeScript Configuration

Add LiqUIdify types to your tsconfig.json:

```json
{
  "compilerOptions": {
    "types": ["liquidify"],
    "moduleResolution": "bundler",
    "allowSyntheticDefaultImports": true
  }
}
```

### Bundle Optimization

#### 1. Tree-shaking specific components

```tsx
// Only import what you need
import { GlassButton } from "liquidify/button";
import { GlassCard } from "liquidify/card";

// Avoid full library imports in production
// import * as Liquid from 'liquidify' // ‚ùå Imports everything
```

#### 2. Code splitting by route

```tsx
// Next.js
const AdminPanel = dynamic(() => import('./AdminPanel'), {
  loading: () => <GlassSkeleton />,
})

// React Router
const AdminPanel = lazy(() => import('./AdminPanel'))

// Usage
<Suspense fallback={<GlassSkeleton />}>
  <AdminPanel />
</Suspense>
```

#### 3. CSS optimization

```tsx
// Import only needed CSS modules
import "liquidify/styles/core"; // Core styles only
import "liquidify/styles/forms"; // Form components styles
// Instead of
import "liquidify/styles"; // All styles
```

### Environment-Specific Configuration

```tsx
// Conditional imports based on environment
const GlassChart = lazy(() =>
  (process.env.NODE_ENV === "production"
    ? import("liquidify/chart")
    : import("liquidify")
  ).then((mod) => ({ default: mod.GlassChart })),
);

// Development-only features
if (process.env.NODE_ENV === "development") {
  import("liquidify/devtools").then(({ enableDevTools }) => {
    enableDevTools();
  });
}
```

## üêõ Troubleshooting

### Common Issues & Solutions

**Issue**: Hydration mismatch in SSR

```tsx
// Solution: Use dynamic imports or ClientOnly wrapper
const ClientComponent = dynamic(() => import("./Component"), {
  ssr: false,
});
```

**Issue**: CSS not loading correctly

```tsx
// Solution: Ensure styles are imported before component usage
import "liquidify/styles"; // First
// Second
```

**Issue**: TypeScript errors with imports

```json
// Solution: Update tsconfig.json
{
  "compilerOptions": {
    "esModuleInterop": true,
    "allowSyntheticDefaultImports": true
  }
}
```

**Issue**: Build errors with older bundlers

```js
// Solution: Transpile the package
// next.config.js
module.exports = {
  transpilePackages: ["liquidify"],
};
```
