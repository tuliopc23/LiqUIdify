## <div className="flex items-center gap-3 text-title-2 mb-6"><SFSymbol name="code" size="xl" color="var(--ios-green)" />Usage Examples</div>

```css
/* Using iOS system colors */
.primary-button {
  background-color: var(--ios-blue);
  color: white;
}

/* Using semantic colors */
.card-title {
  color: var(--apple-label-primary);
}

.card-subtitle {
  color: var(--apple-label-secondary);
}

/* Using liquid glass effects */
.glass-card {
  background: var(--liquid-surface-medium);
  backdrop-filter: blur(var(--glass-blur));
  border: var(--glass-border-width) solid var(--liquid-border-light);
  box-shadow: var(--glass-shadow);
}

/* Combining colors with glass effects */
.success-notification {
  background: var(--liquid-green);
  backdrop-filter: blur(var(--glass-blur));
  border: 1px solid rgba(52, 199, 89, 0.3);
}
```
