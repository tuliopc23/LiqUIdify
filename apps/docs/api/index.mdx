# Public API

This page summarizes the exports from `liquidify` and points to common props and patterns.

## Providers

- `GlassUIProvider` — Wraps your app and orchestrates Theme, Liquid Glass, Haptics, Live Region and Toast.

```tsx
;
import "liquidify/styles";

export default function App({ children }) {
  return (
    <GlassUIProvider theme="light" glassConfig={{ blur: 4, saturation: 120 }}>
      {children}
    </GlassUIProvider>
  );
}
```

Props:

- `theme`: "light" | "dark" (default: "light")
- `glassConfig`: Configuration object for liquid glass effects
- `hapticConfig`: Configuration object for haptic feedback

## Hooks

- `useTheme()` — Returns `{ theme, setTheme }`.
- `useLiquidGlass()` — Returns liquid-glass runtime configuration and helpers.
- `useHapticFeedback()` — Trigger subtle feedback on interactions.
- `useSsrSafeHooks()` / `useSsrSafe()` — Guards for SSR.
- `usePerformanceMonitoring()` — Perf insight utilities for components.

Example (live):

```tsx
;

export default function ThemeToggle() {
  const { theme, setTheme } = useTheme();
  return (
    <GlassButton
      variant="ghost"
      onClick={() => setTheme(theme === "light" ? "dark" : "light")}
    >
      Toggle: {theme}
    </GlassButton>
  );
}
```

## Components (selected)

- Buttons: `GlassButton`, `GlassResponsiveButton`
- Cards: `GlassCard` (refactored), `GlassResponsiveCard`
- Inputs: `GlassInput`, `GlassCheckbox`, `GlassRadioGroup`, `GlassSelect`, `GlassCombobox`, `GlassNumberInput`, `GlassTextarea`, `GlassFormField`
- Feedback: `GlassToast`, `GlassSpinner`, `GlassProgress`, `GlassSkeleton`, `GlassAlert`, `GlassNotification`
- Layout/Navigation: `GlassModal`, `GlassDrawer`, `GlassDropdown`, `Navbar`, `Sidebar`, `GlassBreadcrumbs`, `GlassMobileNav`
- Data: `GlassTable`, `GlassTabs`, `GlassTimeline`, `GlassTooltip`, `GlassPopover`, `GlassTreeView`, `GlassChart`, `GlassPagination`
- Accessibility: `GlassErrorBoundary`, `GlassFocusTrap`, `GlassSkipNavigation`, `GlassLiveRegion`

Import and usage are consistent:

```tsx
;

<GlassButton variant="primary" size="lg">
  Label
</GlassButton>;
```

For any component props, check your editor’s TypeScript hints — types ship with the package.
