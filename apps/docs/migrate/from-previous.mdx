# Migration Guide

This guide helps you migrate between different versions of LiqUIdify, ensuring a smooth transition while taking advantage of new features and improvements.

## Current Version: 2.0.0

LiqUIdify 2.0 introduces significant improvements in performance, accessibility, and developer experience. This migration guide covers the breaking changes and how to update your code.

## Breaking Changes

### 1. Import Structure Changes

**Before (v1.x):**

```tsx
import { Button, Card, Input } from "liquidify";
import { useTheme } from "liquidify/hooks";
import "liquidify/styles";
```

**After (v2.x):**

```tsx
import { GlassButton, GlassCard, GlassInput } from "liquidify";
import { useGlassTheme } from "liquidify";
import "liquidify/css";
```

**Migration:**

```tsx
// Automated migration with codemod
npx @liquidify/migrate v1-to-v2 src/

// Manual find & replace patterns
// Button -> GlassButton
// Card -> GlassCard
// Input -> GlassInput
// useTheme -> useGlassTheme
```

### 2. Provider Changes

**Before (v1.x):**

```tsx
import { LiquidifyProvider } from "liquidify";

function App() {
  return (
    <LiquidifyProvider theme="dark">
      <YourApp />
    </LiquidifyProvider>
  );
}
```

**After (v2.x):**

```tsx
import { UnifiedGlassProvider } from "liquidify";

function App() {
  return (
    <UnifiedGlassProvider defaultTheme="dark" storageKey="app-theme">
      <YourApp />
    </UnifiedGlassProvider>
  );
}
```

### 3. Component Prop Changes

#### Button Component

**Before (v1.x):**

```tsx
<Button
  isLoading={loading}
  loadingText="Please wait..."
  variant="primary"
  size="medium"
/>
```

**After (v2.x):**

```tsx
<GlassButton
  loading={loading}
  loadingText="Please wait..."
  variant="primary"
  size="md"
/>
```

#### Size Prop Standardization

All components now use consistent size values:

| v1.x          | v2.x |
| ------------- | ---- |
| `small`       | `sm` |
| `medium`      | `md` |
| `large`       | `lg` |
| `extra-large` | `xl` |

### 4. Event Handler Changes

**Before (v1.x):**

```tsx
<Button onPress={handlePress} />
<Input onChange={(value) => setValue(value)} />
```

**After (v2.x):**

```tsx
<GlassButton onClick={handlePress} />
<GlassInput onChange={(e) => setValue(e.target.value)} />
```

### 5. CSS Variable Updates

**Before (v1.x):**

```css
:root {
  --lq-primary: #3b82f6;
  --lq-glass-blur: 10px;
  --lq-border-radius: 8px;
}
```

**After (v2.x):**

```css
:root {
  --glass-primary: rgba(59, 130, 246, 0.2);
  --glass-blur: blur(16px);
  --glass-radius: 12px;
}
```

## Step-by-Step Migration

### Step 1: Update Dependencies

```bash
# Remove old version
npm uninstall liquidify

# Install new version
npm install liquidify@latest

# For TypeScript projects
npm install @types/react@^18.0.0
```

### Step 2: Run Automated Migration

LiqUIdify provides a codemod for automated migration:

```bash
# Install migration tool
npm install -g @liquidify/migrate

# Run migration (dry run first)
npx @liquidify/migrate v1-to-v2 src/ --dry-run

# Apply changes
npx @liquidify/migrate v1-to-v2 src/
```

### Step 3: Update Provider

Replace your root provider:

```tsx
// Before
import { LiquidifyProvider } from "liquidify";

// After
import { UnifiedGlassProvider } from "liquidify";

function App() {
  return <UnifiedGlassProvider>{/* Your app */}</UnifiedGlassProvider>;
}
```

### Step 4: Update Imports

Update component imports throughout your codebase:

```tsx
// Use find & replace in your IDE
// Find: from '@liquidify/components'
// Replace: from 'liquidify'

// Update component names
// Button -> GlassButton
// Card -> GlassCard
// Input -> GlassInput
// etc.
```

### Step 5: Update CSS Imports

```tsx
// Before
import "liquidify/styles";
import "liquidify/theme.css";

// After
import "liquidify/css";
```

### Step 6: Fix Breaking Changes

Review and fix any remaining breaking changes:

1. **Component props**: Update changed prop names
2. **Event handlers**: Update event handler signatures
3. **CSS variables**: Update custom CSS variables
4. **TypeScript types**: Update type imports

## Troubleshooting Common Issues

### Build Errors

**CSS Import Issues:**

```tsx
// If you see CSS import errors
// Make sure you have the correct import
import "liquidify/css";

// For Vite projects, you may need
import "liquidify/dist/index.css";
```

**TypeScript Errors:**

```bash
# Clear TypeScript cache
rm -rf node_modules/.cache
npm install

# Update TypeScript config
# Ensure "moduleResolution": "bundler" in tsconfig.json
```

**Bundle Size Issues:**

```tsx
// Ensure you're using tree-shakeable imports
import { GlassButton } from "liquidify/core";
// instead of
import { GlassButton } from "liquidify";
```

### Runtime Errors

**Provider Not Found:**

```tsx
// Make sure UnifiedGlassProvider wraps your app
function App() {
  return <UnifiedGlassProvider>{/* Your components */}</UnifiedGlassProvider>;
}
```

**Theme Issues:**

```tsx
// Check that theme is properly configured
<UnifiedGlassProvider defaultTheme="system" storageKey="theme">
  <App />
</UnifiedGlassProvider>
```
