# Theming

LiqUIdify uses a Liquid Glass token system via CSS variables and a Tailwind plugin. Customize tokens at the app level while components remain transparent and layered.

## Core Tokens

```css
:root {
  /* Decorative vs readable surfaces */
  --lg-bg-color: rgba(255, 255, 255, 0.35); /* decorative */
  --lg-bg-readable: rgba(20, 20, 25, 0.65); /* high contrast surface */
  --lg-highlight: rgba(255, 255, 255, 0.75);
  --lg-text: #1a1d21;
  --lg-text-inverse: #ffffff;
  --lg-accent: #fb4268;
  --lg-grey: #444739;
}

.dark {
  --lg-bg-color: rgba(20, 20, 25, 0.35);
  --lg-bg-readable: rgba(20, 20, 25, 0.75);
  --lg-text: #ffffff;
  --lg-text-inverse: #1a1d21;
}
```

## Radius, Shadows, Filters

```css
:root {
  --lg-radius-md: 1.25rem;
  --lg-shadow-main: 0 6px 6px rgba(0, 0, 0, 0.2), 0 0 20px rgba(0, 0, 0, 0.1);
  --lg-specular-shadow:
    inset 1px 1px 0 rgba(255, 255, 255, 0.75),
    inset 0 0 5px rgba(255, 255, 255, 0.75);
  --lg-filter-blur: blur(4px);
  --lg-filter-enhance: saturate(120%) brightness(1.15);
}
```

## Provider and Mode

```tsx
import { GlassUIProvider } from "liquidify";
import "liquidify/styles";

export function App({ children }) {
  return (
    <GlassUIProvider theme="light" glassConfig={{ blur: 4, saturation: 120 }}>
      {children}
    </GlassUIProvider>
  );
}
```

Toggle theme at runtime:

```tsx live
import { useTheme, GlassButton } from "liquidify";

export default function Toggle() {
  const { theme, setTheme } = useTheme();
  return (
    <GlassButton
      variant="ghost"
      onClick={() => setTheme(theme === "light" ? "dark" : "light")}
    >
      Current: {theme}
    </GlassButton>
  );
}
```

Changes are non-destructive: components keep their transparency and depth, while your page background can vary per brand.
