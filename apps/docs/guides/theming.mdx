# Advanced Theming & Customization

Liquidify features a comprehensive theming system that enables deep customization of the liquid glass aesthetic while maintaining Apple HIG design principles and accessibility standards.

## Quick Start: Programmatic Theme Control

Use the library helpers to control the theme at runtime via the `data-theme` attribute. The value persists to `localStorage`.

```tsx
import { initTheme, setTheme, getTheme } from "liquidify";
import "liquidify/styles";

// Initialize on app startup (reads localStorage or system preference)
initTheme({ storageKey: "liquidify-theme" });

// Example: simple toggle handler
function toggleTheme() {
  const next = getTheme() === "light" ? "dark" : "light";
  setTheme(next, { storageKey: "liquidify-theme" });
}
```

These helpers update `document.documentElement.dataset.theme` to `"light" | "dark"` and keep class names (`light`, `dark`, `theme-light`, `theme-dark`) in sync for compatibility.

## üé® Theme Architecture Overview

### Core Theme Provider Setup

The `GlassUIProvider` serves as the foundation for all theming capabilities:

```tsx
import { GlassUIProvider } from "@liquidify/react";
import "@liquidify/react/styles";

export default function App({ children }) {
  return (
    <GlassUIProvider
      theme="auto"                    // 'light' | 'dark' | 'auto'
      glassConfig={{
        intensity: 0.8,               // Global glass effect intensity (0-1)
        blur: 25,                     // Backdrop blur radius (px)
        saturation: 1.3,              // Color saturation multiplier
        brightness: 1.1,              // Brightness adjustment
        adaptToContent: true          // Content-aware glass adaptation
      }}
      motionConfig={{
        springTension: 400,           // Physics animation tension
        springFriction: 30,           // Physics animation friction
        reduceMotion: false           // Respect user motion preferences
      }}
      accessibilityConfig={{
        announceThemeChanges: true,   // Screen reader announcements
        highContrastMode: false,      // Enhanced contrast mode
        focusRingWidth: 2            // Focus indicator thickness
      }}
    >
      {children}
    </GlassUIProvider>
  );
}
```

### Theme Context Integration

Access and modify theme state throughout your application:

```tsx
import { useTheme } from "@liquidify/react";

function ThemeControls() {
  const {
```

## üåä Liquid Glass Customization

### Component-Level Glass Control

Fine-tune glass effects on individual components:

```tsx
// High-intensity glass for hero sections
<GlassCard 
  glassConfig={{ 
    intensity: 0.9, 
    blur: 30,
    saturation: 1.4 
  }}
  className="hero-section"
>
  <h1>Premium Content</h1>
</GlassCard>

// Subtle glass for dense content areas
<GlassCard 
  glassConfig={{ 
    intensity: 0.4, 
    blur: 15,
    saturation: 1.1 
  }}
  className="content-area"
>
  <article>Readable text content...</article>
</GlassCard>
```

### Content-Aware Glass Adaptation

Enable automatic glass adjustment based on underlying content:

```tsx
<GlassUIProvider
  glassConfig={{
    adaptToContent: true,
    contentSampling: 'intelligent',  // 'simple' | 'intelligent' | 'manual'
    adaptationSensitivity: 0.7      // How aggressive the adaptation (0-1)
  }}
>
  {/* Glass components automatically adjust based on background */}
</GlassUIProvider>
```

### Theme Persistence

```tsx
import { GlassUIProvider, createThemeStorage } from "@liquidify/react";

// Custom theme persistence
const themeStorage = createThemeStorage({
  key: 'liquidify-theme',
  storage: localStorage,  // or sessionStorage, or custom storage
  serialize: JSON.stringify,
  deserialize: JSON.parse
});

function App() {
  return (
    <GlassUIProvider
      themeStorage={themeStorage}
      theme="auto"
    >
      <AppContent />
    </GlassUIProvider>
  );
}
```
