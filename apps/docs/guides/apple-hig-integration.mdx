## <span className="sf-icon-text"><span className="sf-icon sf-star"></span>Core Design Principles</span>

Liquidify embodies Apple's fundamental design principles through its liquid glass design language:

<CardGroup cols={2}>
  <Card title="Clarity" icon="eye">
    **Visual Hierarchy**: Clear typography scales, purposeful contrast, and strategic 
    use of transparency ensure content remains the primary focus while glass effects 
    enhance rather than distract.
  </Card>

  <Card title="Deference" icon="layers">
    **Content First**: Liquid glass materials create sophisticated backgrounds that 
    defer to content, using subtle blur and transparency to establish context without 
    competing for attention.
  </Card>

  <Card title="Depth" icon="cube">
    **Spatial Relationships**: Multi-layered glass effects create realistic depth 
    perception, establishing clear information hierarchy through z-axis positioning 
    and shadow systems.
  </Card>

  <Card title="Consistency" icon="grid">
    **Predictable Interactions**: Familiar Apple interaction patterns combined with 
    physics-based animations create intuitive user experiences across all platforms.
  </Card>
</CardGroup>

## <span className="sf-icon-text"><span className="sf-icon sf-book"></span>Best Practices</span>

<AccordionGroup>
  <Accordion title="Typography" icon="text">
    - Use the Apple HIG type scale consistently - Maintain proper line heights
    and letter spacing - Ensure sufficient contrast ratios (4.5:1 minimum) - Use
    SF Pro fonts when available
  </Accordion>

<Accordion title="Color Usage" icon="palette">
  - Stick to the semantic color system - Use Apple Blue (#0071e3) for primary
  actions - Implement proper dark mode support - Test colors in both light and
  dark themes
</Accordion>

<Accordion title="Spacing" icon="grid">
  - Follow the 8pt grid system religiously - Use consistent spacing between
  related elements - Maintain proper touch targets (44px minimum) - Create
  visual hierarchy with spacing
</Accordion>

  <Accordion title="Glass Effects" icon="circle">
    - Use backdrop blur sparingly for emphasis - Ensure sufficient contrast
    behind glass elements - Test glass effects on various backgrounds - Provide
    fallbacks for unsupported browsers
  </Accordion>
</AccordionGroup>
