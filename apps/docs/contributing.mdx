title: Contributing
description: Dev scripts, PR checklist, and conventions.

See the repository guide:

- [CONTRIBUTING.md](https://github.com/tuliopc23/LiqUIdify/blob/main/CONTRIBUTING.md)

Quick start:

```bash
bun install
bun run dev
bun run storybook
```

Checklist:

- TypeScript passes: `bun run type-check`
- Lint/format: `bun run lint`
- Tests: `bun run test:all`
- Update docs when adding components.
