# Theming

LiqUIdify provides a powerful theming system using CSS variables for easy customization.

## Default Theme

The default theme provides a modern glassmorphism design with blue accents:

```css
:root {
  --glass-primary: 59 130 246;
  --glass-secondary: 107 114 128;
  --glass-background: 255 255 255;
  --glass-backdrop: 0.1;
  --glass-blur: 12px;
}
```

## Custom Colors

Override CSS variables to customize colors:

```css
:root {
  --glass-primary: 168 85 247; /* Purple */
  --glass-secondary: 34 197 94; /* Green */
  --glass-accent: 249 115 22; /* Orange */
}
```

## Dark Mode

LiqUIdify automatically adapts to dark mode when you use the provider:

```tsx
<UnifiedGlassProvider theme="dark">
  <App />
</UnifiedGlassProvider>
```

## Glass Effects

Customize the glass effect intensity:

```css
:root {
  --glass-backdrop: 0.2; /* More opaque */
  --glass-blur: 20px; /* More blur */
  --glass-border: 0.5; /* Border opacity */
}
```

For more theming options, see the [Advanced Theming Guide](/guides/advanced-theming).
