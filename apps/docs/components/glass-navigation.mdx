# Glass Navigation

Beautiful glassmorphism navigation components that provide seamless user experience across all devices with responsive design and accessibility.

## Props

### GlassNavbar Props

<ResponseField name="variant" type="string" default="default">
  The visual style of the navbar
  - `"default"` - Standard glass effect
  - `"elevated"` - Enhanced depth and shadows  
  - `"minimal"` - Subtle glass effect
  - `"ghost"` - Ultra-minimal appearance
</ResponseField>

<ResponseField name="sticky" type="boolean" default="false">
  Makes the navbar stick to the top when scrolling
</ResponseField>

<ResponseField name="bordered" type="boolean" default="false">
  Adds a border to the navbar
</ResponseField>

<ResponseField name="blurred" type="boolean" default="true">
  Enables backdrop blur effect
</ResponseField>

### GlassNavItem Props

<ResponseField name="href" type="string">
  URL to navigate to (for link items)
</ResponseField>

<ResponseField name="active" type="boolean" default="false">
  Whether the nav item is currently active/selected  
</ResponseField>

<ResponseField name="disabled" type="boolean" default="false">
  Whether the nav item is disabled
</ResponseField>

<ResponseField name="variant" type="string" default="default">
  Visual style variant
  - `"default"` - Standard appearance
  - `"minimal"` - Minimal styling
  - `"ghost"` - Ultra-subtle appearance
</ResponseField>

<ResponseField name="fullWidth" type="boolean" default="false">
  Makes the item take full width (useful for mobile)
</ResponseField>

<ResponseField name="leftIcon" type="ReactNode">
  Icon to display on the left side
</ResponseField>

<ResponseField name="rightIcon" type="ReactNode">
  Icon to display on the right side
</ResponseField>

<ResponseField name="rightContent" type="ReactNode">
  Custom content to display on the right (badges, etc.)
</ResponseField>

<ResponseField name="as" type="string" default="a">
  Element type to render as
  - `"a"` - Link element
  - `"button"` - Button element  
  - Custom component
</ResponseField>

### GlassMobileNav Props

<ResponseField name="open" type="boolean" default="false">
  Controls visibility of mobile navigation
</ResponseField>

<ResponseField name="onClose" type="() => void">
  Called when mobile nav should close
</ResponseField>

<ResponseField name="side" type="string" default="right">
  Which side the mobile nav slides in from
  - `"left"` - Slide from left
  - `"right"` - Slide from right
  - `"top"` - Slide from top
  - `"bottom"` - Slide from bottom
</ResponseField>

## Accessibility

LiqUIdify navigation components follow accessibility best practices:

- **Semantic HTML**: Uses proper `nav`, `ul`, `li` structure
- **ARIA Labels**: Comprehensive ARIA support for screen readers
- **Keyboard Navigation**: Full keyboard support with Tab, Enter, and arrow keys
- **Focus Management**: Clear focus indicators and logical focus order
- **Mobile Accessibility**: Touch-friendly targets and gestures
- **Screen Reader Support**: Proper announcements for navigation state

### Accessibility Example

```tsx
<GlassNavbar 
  role="navigation" 
  aria-label="Main navigation"
>
  <nav>
    <ul className="flex space-x-4">
      <li>
        <GlassNavItem 
          href="/" 
          active 
          aria-current="page"
          aria-label="Home page, current page"
        >
          Home
        </GlassNavItem>
      </li>
      <li>
        <GlassNavItem 
          href="/about"
          aria-label="About page"
        >
          About
        </GlassNavItem>
      </li>
    </ul>
  </nav>
</GlassNavbar>
```

## Theming

Customize navigation appearance using CSS variables:

```css
.custom-glass-nav {
  --glass-nav-bg: rgba(255, 255, 255, 0.1);
  --glass-nav-border: rgba(255, 255, 255, 0.2);
  --glass-nav-backdrop: blur(16px);
  
  /* Navigation item styles */
  --glass-nav-item-hover: rgba(255, 255, 255, 0.1);
  --glass-nav-item-active: rgba(255, 255, 255, 0.2);
  --glass-nav-item-text: rgb(255, 255, 255);
  
  /* Mobile navigation */
  --glass-mobile-nav-bg: rgba(0, 0, 0, 0.8);
  --glass-mobile-nav-backdrop: blur(20px);
}
```

## Performance Notes

- Navigation components are optimized with `React.memo`
- Glass effects use GPU acceleration for smooth scrolling
- Mobile navigation uses CSS transforms for 60fps animations
- Icons should be properly sized (16px recommended)
- Large navigation menus benefit from virtualization
