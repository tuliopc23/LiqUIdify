## Props

| Prop           | Type                           | Description           |
| -------------- | ------------------------------ | --------------------- |
| `open`         | `boolean`                      | Controlled visibility |
| `onOpenChange` | `(open:boolean)=>void`         | Change callback       |
| `items`        | `{label:string;href:string}[]` | Nav links             |
