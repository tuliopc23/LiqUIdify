---
title: Skip Navigation
description: Provide a quick link to skip repetitive navigation.
---

Overview
Skip Navigation offers a keyboard-accessible link that jumps to main content.

Import
```ts
import { GlassSkipNavigation } from '@liquidify/components'
```

Usage
```tsx
<GlassSkipNavigation targetId="main-content" />

<main id="main-content">
  ...
</main>
```

Accessibility
- Place as the first focusable element in the page.
- Ensure targetId points to a valid landmark (#main, #content, etc.).
