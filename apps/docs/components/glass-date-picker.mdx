## Overview

The Glass Date Picker component provides an elegant date selection interface with Apple-inspired liquid glass effects. It supports date-only and date-time selection with comprehensive customization options.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `Date` | - | Current selected date |
| `defaultValue` | `Date` | - | Default selected date (uncontrolled) |
| `onChange` | `(date: Date \| null) => void` | - | Callback when date changes |
| `placeholder` | `string` | `'Select date...'` | Placeholder text |
| `disabled` | `boolean` | `false` | Disabled state |
| `showTime` | `boolean` | `false` | Enable time selection |
| `format` | `string` | - | Custom date format |
| `locale` | `string` | `'en-US'` | Locale for date formatting |
| `minDate` | `Date` | - | Minimum selectable date |
| `maxDate` | `Date` | - | Maximum selectable date |
| `disabledDates` | `Date[]` | `[]` | Array of disabled dates |
| `weekStartsOn` | `0 \| 1 \| 2 \| 3 \| 4 \| 5 \| 6` | `0` | First day of week (0 = Sunday) |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size variant |

### Form Integration

```tsx
const [formData, setFormData] = useState({
  startDate: null,
  endDate: null,
  reminderDate: null
})

const [errors, setErrors] = useState({})

const handleStartDateChange = (date) => {
  setFormData(prev => ({ ...prev, startDate: date }))
  
  // Clear end date if it's before start date
  if (formData.endDate && date && formData.endDate < date) {
    setFormData(prev => ({ ...prev, endDate: null }))
  }
  
  // Clear error
  if (errors.startDate) {
    setErrors(prev => ({ ...prev, startDate: '' }))
  }
}

<form className="space-y-4">
  <div>
    <label className="block text-sm font-medium mb-2">Start Date</label>
    <GlassDatePicker
      value={formData.startDate}
      onChange={handleStartDateChange}
      placeholder="Select start date..."
      minDate={new Date()}
    />
    {errors.startDate && (
      <p className="text-red-400 text-sm mt-1">{errors.startDate}</p>
    )}
  </div>
  
  <div>
    <label className="block text-sm font-medium mb-2">End Date</label>
    <GlassDatePicker
      value={formData.endDate}
      onChange={(date) => setFormData(prev => ({ ...prev, endDate: date }))}
      placeholder="Select end date..."
      minDate={formData.startDate || new Date()}
    />
  </div>
</form>
```

### Validation and Error Handling

```tsx
const validateDate = (date, minDate, maxDate, disabledDates) => {
  if (!date) return null
  
  if (minDate && date < minDate) {
    return `Date must be after ${minDate.toLocaleDateString()}`
  }
  
  if (maxDate && date > maxDate) {
    return `Date must be before ${maxDate.toLocaleDateString()}`
  }
  
  if (disabledDates.some(disabled => 
    date.toDateString() === disabled.toDateString()
  )) {
    return 'This date is not available'
  }
  
  return null
}

const handleDateChange = (date) => {
  const error = validateDate(date, minDate, maxDate, disabledDates)
  
  if (error) {
    setError(error)
    return
  }
  
  setError('')
  onChange(date)
}
```
