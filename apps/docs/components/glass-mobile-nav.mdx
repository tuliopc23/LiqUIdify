## Overview

The Glass Mobile Nav component provides an elegant mobile navigation solution with Apple-inspired liquid glass effects. It features a hamburger menu that opens a sliding drawer with smooth animations and nested navigation support.

## Props

| Prop               | Type                      | Default                  | Description                              |
| ------------------ | ------------------------- | ------------------------ | ---------------------------------------- |
| `items`            | `NavItem[]`               | -                        | **Required.** Array of navigation items  |
| `activeItemId`     | `string`                  | -                        | ID of the currently active item          |
| `onItemClick`      | `(item: NavItem) => void` | -                        | Callback when an item is clicked         |
| `position`         | `'left' \| 'right'`       | `'right'`                | Position of the drawer                   |
| `showOverlay`      | `boolean`                 | `true`                   | Whether to show backdrop overlay         |
| `closeOnItemClick` | `boolean`                 | `true`                   | Close drawer when item is clicked        |
| `header`           | `ReactNode`               | -                        | Custom header content                    |
| `footer`           | `ReactNode`               | -                        | Custom footer content                    |
| `hamburgerLabel`   | `string`                  | `'Open navigation menu'` | Accessibility label for hamburger button |
| `size`             | `'sm' \| 'md' \| 'lg'`    | `'md'`                   | Size variant                             |

### NavItem Interface

```typescript
interface NavItem {
  id: string;
  label: string;
  href?: string;
  onClick?: () => void;
  icon?: React.ReactNode;
  badge?: string | number;
  disabled?: boolean;
  children?: Array<NavItem>;
}
```

## Usage Patterns

### Basic Navigation

```tsx
const items = [
  { id: 'home', label: 'Home', icon: <Home /> },
  { id: 'about', label: 'About', icon: <Info /> }

<GlassMobileNav items={items} />
```

## Accessibility

### WCAG Compliance

- **Keyboard Navigation**: Full keyboard support with arrow keys, Enter, and Escape
- **Screen Readers**: Proper ARIA labels, roles, and states
- **Focus Management**: Automatic focus handling when opening/closing
- **High Contrast**: Supports system color preferences

### Implementation Details

- Uses `role="navigation"` and `aria-labelledby`
- Hamburger button has `aria-expanded` and `aria-controls`
- Navigation items use `aria-current="page"` for active state
- Expandable items have `aria-expanded` attribute
- Escape key closes the drawer and returns focus

## Apple HIG Guidelines

### Visual Design

- **Liquid Glass Effects**: Multi-layered transparency with blur and specular highlights
- **Smooth Animations**: Spring-based animations with proper easing curves
- **Typography**: SF Pro font family with appropriate weights

- **Magnetic Hover**: Subtle hover effects with micro-interactions

- **Gesture Support**: Swipe gestures for drawer interaction

### Color System

- **Adaptive Colors**: Responds to light/dark mode preferences

- **Contrast Ratios**: Maintains WCAG AA compliance
- **Glass Tinting**: Subtle color overlays for depth

## Best Practices

### Performance

- Use `React.memo` for navigation items that don't change frequently
- Implement virtualization for very long navigation lists
- Lazy load nested navigation content when possible

### UX Guidelines

- Keep navigation hierarchy shallow (max 2-3 levels)
- Use clear, descriptive labels
- Group related items logically
- Provide visual feedback for all interactions

### Responsive Design

- Component automatically adapts to different screen sizes
- Drawer width adjusts based on viewport
- Touch targets meet minimum 44px requirement
- Supports both portrait and landscape orientations
