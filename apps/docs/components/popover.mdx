# Popover

The `GlassPopover` component provides floating content containers with glassmorphism effects. It features intelligent positioning, viewport awareness, and comprehensive interaction handling.

## Live Examples

### Basic Popover

```tsx live
import { GlassPopover, GlassButton } from "@liquidify/components";

export default function Demo() {
  return (
    <div
      style={{
        display: "flex",
        gap: 16,
        justifyContent: "center",
        padding: 40,
      }}
    >
      <GlassPopover
        trigger={<GlassButton>Click me</GlassButton>}
        content={
          <div>
            <h4 style={{ margin: "0 0 8px 0" }}>Popover Content</h4>
            <p style={{ margin: 0, fontSize: "0.875rem", opacity: 0.8 }}>
              This is a basic popover with glassmorphism effects.
            </p>
          </div>
        }
      />
    </div>
  );
}
```

### Controlled Popover

```tsx live
import { useState } from "react";
import { GlassPopover, GlassButton } from "@liquidify/components";

export default function Demo() {
  const [open, setOpen] = useState(false);

  return (
    <div
      style={{
        display: "flex",
        flexDirection: "column",
        gap: 16,
        alignItems: "center",
        padding: 40,
      }}
    >
      <div style={{ display: "flex", gap: 12 }}>
        <GlassButton onClick={() => setOpen(true)}>Open Popover</GlassButton>
        <GlassButton variant="ghost" onClick={() => setOpen(false)}>
          Close Popover
        </GlassButton>
      </div>

      <GlassPopover
        open={open}
        onOpenChange={setOpen}
        trigger={<GlassButton>Controlled Trigger</GlassButton>}
        content={
          <div>
            <h4 style={{ margin: "0 0 8px 0" }}>Controlled Popover</h4>
            <p
              style={{
                margin: "0 0 12px 0",
                fontSize: "0.875rem",
                opacity: 0.8,
              }}
            >
              This popover's state is controlled externally.
            </p>
            <GlassButton size="sm" onClick={() => setOpen(false)}>
              Close
            </GlassButton>
          </div>
        }
      />
    </div>
  );
}
```

### Form in Popover

```tsx live
import { useState } from "react";
import { GlassPopover, GlassButton } from "@liquidify/components";

export default function Demo() {
  const [formData, setFormData] = useState({ name: "", email: "" });
  const [open, setOpen] = useState(false);

  const handleSubmit = (e) => {
    e.preventDefault();
    alert(`Submitted: ${formData.name} (${formData.email})`);
    setOpen(false);
    setFormData({ name: "", email: "" });
  };

  return (
    <div style={{ display: "flex", justifyContent: "center", padding: 40 }}>
      <GlassPopover
        open={open}
        onOpenChange={setOpen}
        trigger={<GlassButton>Quick Signup</GlassButton>}
        content={
          <form onSubmit={handleSubmit} style={{ minWidth: 240 }}>
            <h4 style={{ margin: "0 0 16px 0" }}>Quick Signup</h4>

            <div style={{ marginBottom: 12 }}>
              <label
                style={{
                  display: "block",
                  marginBottom: 4,
                  fontSize: "0.875rem",
                }}
              >
                Name
              </label>
              <input
                type="text"
                value={formData.name}
                onChange={(e) =>
                  setFormData((prev) => ({ ...prev, name: e.target.value }))
                }
                style={{
                  width: "100%",
                  padding: "8px 12px",
                  border: "1px solid rgba(255,255,255,0.2)",
                  borderRadius: "8px",
                  background: "rgba(255,255,255,0.1)",
                  color: "white",
                  fontSize: "0.875rem",
                }}
                required
              />
            </div>

            <div style={{ marginBottom: 16 }}>
              <label
                style={{
                  display: "block",
                  marginBottom: 4,
                  fontSize: "0.875rem",
                }}
              >
                Email
              </label>
              <input
                type="email"
                value={formData.email}
                onChange={(e) =>
                  setFormData((prev) => ({ ...prev, email: e.target.value }))
                }
                style={{
                  width: "100%",
                  padding: "8px 12px",
                  border: "1px solid rgba(255,255,255,0.2)",
                  borderRadius: "8px",
                  background: "rgba(255,255,255,0.1)",
                  color: "white",
                  fontSize: "0.875rem",
                }}
                required
              />
            </div>

            <div
              style={{ display: "flex", gap: 8, justifyContent: "flex-end" }}
            >
              <GlassButton
                type="button"
                variant="ghost"
                size="sm"
                onClick={() => setOpen(false)}
              >
                Cancel
              </GlassButton>

                Sign Up
```

## Props

| Prop                  | Type                                     | Default     | Description                            |
| --------------------- | ---------------------------------------- | ----------- | -------------------------------------- |
| `trigger`             | `React.ReactNode`                        | -           | Element that triggers the popover      |
| `content`             | `React.ReactNode`                        | -           | Content to display in the popover      |
| `position`            | `"top" \| "bottom" \| "left" \| "right"` | `"bottom"`  | Preferred position relative to trigger |
| `align`               | `"start" \| "center" \| "end"`           | `"center"`  | Alignment relative to the trigger      |
| `open`                | `boolean`                                | `undefined` | Controlled open state                  |
| `onOpenChange`        | `(open: boolean) => void`                | `undefined` | Callback when open state changes       |
| `closeOnClickOutside` | `boolean`                                | `true`      | Close when clicking outside            |
| `closeOnEscape`       | `boolean`                                | `true`      | Close when pressing Escape key         |
| `className`           | `string`                                 | `undefined` | Additional CSS classes for trigger     |
| `contentClassName`    | `string`                                 | `undefined` | Additional CSS classes for content     |

## Usage Patterns

### Menu Popover

```tsx
const MenuPopover = () => {
  const menuItems = [
    { label: "Edit", action: () => console.log("Edit") },
    { label: "Delete", action: () => console.log("Delete") },
    { label: "Share", action: () => console.log("Share") },
  ];

  return (
    <GlassPopover
      trigger={<GlassButton>Actions</GlassButton>}
      content={
        <div className="py-1">
          {menuItems.map((item) => (
            <button
              key={item.label}
              onClick={item.action}
              className="block w-full px-4 py-2 text-left hover:bg-white/10"
            >
              {item.label}
            </button>
          ))}
        </div>
      }
    />
  );
};
```

### Confirmation Popover

```tsx
const ConfirmationPopover = ({ onConfirm, children }) => {
  const [open, setOpen] = useState(false);

  return (
    <GlassPopover
      open={open}
      onOpenChange={setOpen}
      trigger={children}
      content={
        <div>
          <p>Are you sure you want to delete this item?</p>
          <div className="flex gap-2 mt-3">
            <GlassButton size="sm" onClick={() => setOpen(false)}>
              Cancel
            </GlassButton>
            <GlassButton
              size="sm"
              variant="destructive"
              onClick={() => {
                onConfirm();
                setOpen(false);
              }}
            >
              Delete
            </GlassButton>
          </div>
        </div>
      }
    />
  );
};
```

### Search Popover

```tsx
const SearchPopover = () => {
  const [query, setQuery] = useState("");
  const [results, setResults] = useState([]);

  useEffect(() => {
    if (query) {
      // Simulate search
      setResults(
        mockData.filter((item) =>
          item.name.toLowerCase().includes(query.toLowerCase()),
        ),
      );
    } else {
      setResults([]);
    }
  }, [query]);

  return (
    <GlassPopover
      trigger={<GlassButton>Search</GlassButton>}
      content={
        <div className="w-80">
          <input
            type="text"
            placeholder="Search..."
            value={query}
            onChange={(e) => setQuery(e.target.value)}
            className="w-full p-2 mb-2 rounded border"
          />
          <div className="max-h-60 overflow-y-auto">
            {results.map((result) => (
              <div key={result.id} className="p-2 hover:bg-white/10">
                {result.name}
              </div>
            ))}
          </div>
        </div>
      }
    />
  );
};
```

## Accessibility

### WCAG Compliance

- **WCAG 2.1 AA**: Full compliance with keyboard navigation and screen readers
- **Keyboard Navigation**: Enter, Space, and Escape key support
- **Screen Readers**: Proper ARIA attributes and announcements
- **Focus Management**: Focus trapping and restoration

### Implementation Details

```tsx
// Accessible popover with proper ARIA
<GlassPopover
  trigger={
    <GlassButton
      aria-haspopup="true"
      aria-expanded={open}
      aria-describedby="popover-content"
    >
      Open Menu
    </GlassButton>
  }
  content={
    <div id="popover-content" role="menu" aria-label="Action menu">
      {/* Menu items */}
    </div>
  }
/>
```

### Best Practices

- Use appropriate ARIA roles for different content types
- Ensure keyboard navigation within popover content
- Provide clear focus indicators
- Include proper labeling for screen readers
- Handle focus management when opening/closing
