# Components Showcase

A curated collection of the most commonly used LiqUIdify components with code examples. For interactive previews, visit our [Storybook](https://liquidify-storybook.vercel.app).

## Button

Multiple button variants with different styles and interactions.

```tsx
import { GlassButton } from "liquidify";

export default function ButtonShowcase() {
  return (
    <div style={{ display: 'flex', gap: 12 }}>
      <GlassButton variant="primary">Primary</GlassButton>
      <GlassButton variant="secondary">Secondary</GlassButton>
      <GlassButton variant="ghost">Ghost</GlassButton>
    </div>
  );
}
```

## Form Field with Input

Complete form field with label, input, and helper text.

```tsx
import { FormField, Input } from "liquidify";

export default function FormFieldShowcase() {
  return (
    <FormField 
      label="Email" 
      helperText="We'll never share your email."
    >
      <Input placeholder="you@example.com" />

```

## Combobox

Searchable dropdown with options filtering.

```tsx
import { Combobox } from "liquidify";

export default function ComboboxShowcase() {
  const options = [
    { label: "Apple", value: "apple" },
    { label: "Banana", value: "banana" },
    { label: "Orange", value: "orange" },
  ];

  return (
    <Combobox 
      options={options} 
      placeholder="Pick a fruit" 
    />
  );
}
```

## Modal

Modal dialog with backdrop and close functionality.

```tsx
import { Modal, GlassButton } from "liquidify";
import { useState } from "react";

export default function ModalShowcase() {
  const [open, setOpen] = useState(false);

  return (
    <div>
      <GlassButton onClick={() => setOpen(true)}>
        Open Modal
      </GlassButton>
      <Modal 
        open={open} 
        onOpenChange={setOpen} 
        title="Hello"
      >
        Modal content goes here.
      </Modal>
    </div>
  );
}
```

## Toast Notifications

Toast system with provider and hook.

```tsx
import { ToastProvider, useToast, GlassButton } from "liquidify";

function ToastDemo() {
  const { toast } = useToast();

  const showToast = () => {
    toast({ 
      title: "Saved", 
      description: "Changes persisted." 
    });
  };

  return (
    <GlassButton onClick={showToast}>
      Show Toast
    </GlassButton>
  );
}

export default function ToastShowcase() {
  return (
    <ToastProvider>
      <ToastDemo />
    </ToastProvider>
  );
}
```
