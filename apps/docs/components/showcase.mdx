# Components Showcase

A curated collection of the most commonly used Liquidify components with live snippet previews (rendered) followed by their code blocks. For more examples, visit our [Storybook](https://liquidify-storybook.vercel.app).

## Overview

Explore a quick visual overview. Each component is presented with a short description, a live preview, and a usage code block.

### Button

A primary action trigger with hover, focus, and active states.

```tsx
import { GlassButton } from "@liquidify/react";

export default function Example() {
  return <GlassButton variant="primary">Click me</GlassButton>;
}
```

### Input

Text input with clear focus and hover states.

```tsx
import { GlassInput } from "@liquidify/react";

export default function Example() {
  return <GlassInput placeholder="Type here" />;
}
```

### Card

Container for grouping related content.

<CardExample />

```tsx title="Usage"
import { Card } from "liquidify";

export default function Example() {
  return (
    <Card title="Card title" subtitle="Subtitle">
      Content
    </Card>
  );
}
```

### Modal

Overlay dialog with accessible focus management.

<ModalExample />

```tsx title="Usage"
import { Modal, Button } from "liquidify";
import React from "react";

export default function Example() {
  const [open, setOpen] = React.useState(false);
  return (
    <>
      <Button variant="primary" onClick={() => setOpen(true)}>
        Open modal
      </Button>
      <Modal open={open} onOpenChange={setOpen} title="Modal title">
        <p>Modal content</p>
      </Modal>
    </>
  );
}
```

### Tabs

Switch between content sections.

<TabsExample />

```tsx title="Usage"
import { Tabs } from "liquidify";

export default function Example() {
  return (
    <Tabs
      tabs={[
        { id: "tab1", label: "Tab One", content: <div>Content One</div> },
        { id: "tab2", label: "Tab Two", content: <div>Content Two</div> },
        { id: "tab3", label: "Tab Three", content: <div>Content Three</div> }
      ]
      initialTabId="tab1"
    />
  );
}
```

### Tooltip

Contextual hint shown on hover or focus.

<TooltipExample />

```tsx title="Usage"
import { Tooltip, Button } from "liquidify";

export default function Example() {
  return (
    <Tooltip content="Helpful tip">
      <Button variant="secondary">Hover me</Button>
    </Tooltip>
  );
}
```

### Toast

Transient notification with optional description.

<ToastExample />

```tsx title="Usage"
import { Toast, Button } from "liquidify";
import React from "react";

export default function Example() {
  const [open, setOpen] = React.useState(false);
  return (
    <>
      <Button onClick={() => setOpen(true)}>Show toast</Button>
      <Toast
        open={open}
        onOpenChange={setOpen}
        title="Saved"
        description="Your changes have been saved."
      />
    </>
  );
}
```
