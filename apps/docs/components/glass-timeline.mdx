---
title: Glass Timeline
description: Display chronological events in a vertical or horizontal list.
---

Overview
Glass Timeline renders events with Liquid Glass styling.

Import
```ts
import { GlassTimeline } from '@liquidify/components'
```

Usage
```tsx
const items = [
  { id: 1, title: 'Created project', date: '2025-01-01' },
  { id: 2, title: 'First release', date: '2025-02-15' },
]

<GlassTimeline items={items} orientation="vertical" />
```

Props
- items: { id: string | number; title: string; date?: string; content?: ReactNode }[]
- orientation?: 'vertical' | 'horizontal' (default: 'vertical')
- renderItem?: (item) => ReactNode
