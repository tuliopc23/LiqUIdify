# Radio Group

A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.

## Installation

```bash
npm install liquidify
```

## Usage

```tsx
import { GlassRadioGroup } from "@liquidify/react";

function Example() {
  const [value, setValue] = useState("option1");

  return (
    <GlassRadioGroup
      value={value}
      onValueChange={setValue}
      options={[
        { value: "option1", label: "Option 1" },
        { value: "option2", label: "Option 2" },
        { value: "option3", label: "Option 3" },
      ]}
    />
  );
}
```

## Props

| Prop            | Type                                    | Default      | Description                             |
| --------------- | --------------------------------------- | ------------ | --------------------------------------- |
| `value`         | `string`                                | -            | The controlled value of the radio group |
| `onValueChange` | `(value: string) => void`               | -            | Callback fired when the value changes   |
| `options`       | `Array<{value: string, label: string}>` | -            | Array of radio options                  |
| `disabled`      | `boolean`                               | `false`      | Whether the radio group is disabled     |
| `orientation`   | `"horizontal" \| "vertical"`            | `"vertical"` | The orientation of the radio group      |

## Examples

### Basic Radio Group

```tsx
<GlassRadioGroup
  value={value}
  onValueChange={setValue}
  options={[
    { value: "small", label: "Small" },
    { value: "medium", label: "Medium" },
    { value: "large", label: "Large" },
  ]}
/>
```

### Horizontal Layout

```tsx
<GlassRadioGroup
  orientation="horizontal"
  value={value}
  onValueChange={setValue}
  options={[
    { value: "left", label: "Left" },
    { value: "center", label: "Center" },
    { value: "right", label: "Right" },
  ]}
/>
```

### Disabled State

```tsx
<GlassRadioGroup
  disabled
  value={value}
  onValueChange={setValue}
  options={[
    { value: "option1", label: "Option 1" },
    { value: "option2", label: "Option 2" },
  ]}
/>
```

## Accessibility

- Uses proper ARIA attributes for screen readers
- Supports keyboard navigation with arrow keys
- Follows WAI-ARIA Radio Group pattern
- Provides focus management and visual indicators
