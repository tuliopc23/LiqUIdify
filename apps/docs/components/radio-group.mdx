## Props

### GlassRadioGroup

| Prop            | Type                         | Default      | Description                     |
| --------------- | ---------------------------- | ------------ | ------------------------------- |
| `value`         | `string`                     | -            | Currently selected value        |
| `onValueChange` | `(value: string) => void`    | -            | Callback when selection changes |
| `orientation`   | `'horizontal' \| 'vertical'` | `'vertical'` | Layout orientation              |
| `size`          | `'sm' \| 'md' \| 'lg'`       | `'md'`       | Size variant                    |
| `disabled`      | `boolean`                    | `false`      | Disabled state                  |
| `name`          | `string`                     | -            | Form name attribute             |
| `required`      | `boolean`                    | `false`      | Required field                  |
| `children`      | `ReactNode`                  | -            | Radio items                     |

### GlassRadioGroup.Item

| Prop       | Type                              | Default     | Description    |
| ---------- | --------------------------------- | ----------- | -------------- |
| `value`    | `string`                          | -           | Item value     |
| `disabled` | `boolean`                         | `false`     | Disabled state |
| `size`     | `'sm' \| 'md' \| 'lg'`            | `'md'`      | Size variant   |
| `variant`  | `'default' \| 'solid' \| 'ghost'` | `'default'` | Visual variant |
| `children` | `ReactNode`                       | -           | Item content   |

## Accessibility

### WCAG Compliance

- **Keyboard Navigation**: Full keyboard support with arrow keys
- **Screen Reader**: Proper ARIA attributes and announcements
- **Focus Management**: Clear focus indicators and logical tab order
- **Grouping**: Proper radio group semantics

### Keyboard Shortcuts

| Key          | Action                         |
| ------------ | ------------------------------ |
| `Tab`        | Move focus to/from radio group |
| `Arrow Keys` | Navigate between radio options |
| `Space`      | Select focused radio option    |

### Screen Reader Support

```tsx
// Automatic ARIA attributes
<GlassRadioGroup value={value} onValueChange={setValue}>
  <GlassRadioGroup.Item value="option1">Option One</GlassRadioGroup.Item>
  <GlassRadioGroup.Item value="option2">Option Two</GlassRadioGroup.Item>
</GlassRadioGroup>

// Generates:
// <div role="radiogroup" aria-labelledby="group-label">
//   <div role="radio" aria-checked="true" tabindex="0">
//     Option One
//   </div>
//   <div role="radio" aria-checked="false" tabindex="-1">
//     Option Two
//   </div>
// </div>
```

### ARIA Attributes

- **`role="radiogroup"`**: Identifies the container as a radio group
- **`role="radio"`**: Identifies each option as a radio button
- **`aria-checked`**: Indicates the checked state of each radio
- **`aria-labelledby`**: Associates the group with its label
- **`aria-describedby`**: Links to helper text when provided
- **`tabindex`**: Manages focus within the group (0 for selected, -1 for others)

## Apple HIG Guidelines

### Visual Design

- **Clarity**: Clear visual distinction between selected and unselected states
- **Depth**: Subtle glass effects that enhance the selection metaphor
- **Consistency**: Consistent spacing and alignment across options
- **Hierarchy**: Clear visual hierarchy within rich content items

### Interaction Design

- **Immediate Feedback**: Instant visual response to selection changes
- **Touch Targets**: Appropriate size for touch interaction
- **States**: Clear visual states for normal, hover, focus, and disabled
- **Animations**: Smooth transitions that feel natural and responsive

### Content Organization

- **Logical Grouping**: Related options grouped together
- **Clear Labels**: Descriptive text for each option
- **Progressive Disclosure**: Additional details revealed when appropriate
- **Scannable Layout**: Easy to scan and compare options
