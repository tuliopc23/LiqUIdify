# Avatar

The Avatar component provides a flexible way to display user profile pictures, initials, or placeholder images. It supports various sizes, shapes, and fallback options.

## Basic Usage

```tsx
import { Avatar } from '@liquidify/react';

function BasicAvatar() {
  return (
    <Avatar
      src="/user-avatar.jpg"
      alt="John Doe"
      name="John Doe"
    />
  );
}
```

## Fallback Options

When no image is provided, the avatar shows initials or a placeholder:

```tsx
import { Avatar } from '@liquidify/react';

function AvatarFallbacks() {
  return (
    <div className="flex items-center gap-4">
      {/* Shows initials */}
      <Avatar name="John Doe" />
      
      {/* Shows custom initials */}
      <Avatar name="Jane Smith" initials="JS" />
      
      {/* Shows placeholder icon */}
      <Avatar />
      
      {/* Custom fallback */}
      <Avatar
        fallback={<span>?</span>}
      />
    </div>
  );
}
```

## Avatar Groups

Display multiple avatars in a group:

```tsx
import { Avatar, AvatarGroup } from '@liquidify/react';

function AvatarGroupExample() {
  const users = [
    { name: "John Doe", src: "/user1.jpg" },
    { name: "Jane Smith", src: "/user2.jpg" },
    { name: "Bob Johnson", src: "/user3.jpg" },
    { name: "Alice Brown", src: "/user4.jpg" },
    { name: "Charlie Wilson", src: "/user5.jpg" }
  ];

  return (
    <AvatarGroup max={4}>
      {users.map((user, index) => (
        <Avatar
          key={index}
          src={user.src}
          name={user.name}
        />
      ))}
    </AvatarGroup>
  );
}
```

## Interactive Avatars

Make avatars clickable for user interactions:

```tsx
import { Avatar, Dropdown, DropdownTrigger, DropdownMenu, DropdownItem } from '@liquidify/react';

function InteractiveAvatar() {
  const user = {
    name: "John Doe",
    email: "john@example.com",
    avatar: "/user-avatar.jpg"
  };

  return (
    <Dropdown>
      <DropdownTrigger>
        <Avatar
          src={user.avatar}
          name={user.name}
          className="cursor-pointer hover:opacity-80 transition-opacity"
        />
      </DropdownTrigger>
      <DropdownMenu>
        <DropdownItem>Profile</DropdownItem>
        <DropdownItem>Settings</DropdownItem>
        <DropdownItem>Sign Out</DropdownItem>
      </DropdownMenu>
    </Dropdown>
  );
}
```

## Loading State

Show loading state while avatar image loads:

```tsx
import { Avatar } from '@liquidify/react';

function LoadingAvatar() {
  const [loading, setLoading] = useState(true);

  return (
    <Avatar
      src="/user-avatar.jpg"
      name="John Doe"
      loading={loading}
      onLoad={() => setLoading(false)}
      onError={() => setLoading(false)}
    />
  );
}
```

## API Reference

### Avatar Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `src` | `string` | - | Image source URL |
| `alt` | `string` | - | Image alt text |
| `name` | `string` | - | User name for initials |
| `initials` | `string` | - | Custom initials override |
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Avatar size |
| `shape` | `'circle' \| 'square'` | `'circle'` | Avatar shape |
| `color` | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger' \| string` | `'primary'` | Background color |
| `status` | `'online' \| 'away' \| 'busy' \| 'offline'` | - | Status indicator |
| `loading` | `boolean` | `false` | Loading state |
| `fallback` | `ReactNode` | - | Custom fallback content |
| `className` | `string` | - | Additional CSS classes |
| `style` | `CSSProperties` | - | Inline styles |
| `onClick` | `() => void` | - | Click handler |
| `onLoad` | `() => void` | - | Image load handler |
| `onError` | `() => void` | - | Image error handler |

### AvatarGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `max` | `number` | - | Maximum avatars to show |
| `spacing` | `'sm' \| 'md' \| 'lg'` | `'md'` | Spacing between avatars |
| `className` | `string` | - | Additional CSS classes |
| `children` | `ReactNode` | - | Avatar components |

## Best Practices

### 1. Always Provide Fallbacks

Ensure avatars have proper fallback content:

```tsx
// Good
<Avatar
  src="/user-avatar.jpg"
  name="John Doe"
  alt="John Doe's profile picture"
/>

// Better - with custom fallback
<Avatar
  src="/user-avatar.jpg"
  name="John Doe"
  alt="John Doe's profile picture"
  fallback={<UserIcon />}
/>
```

### 2. Use Appropriate Sizes

Match avatar size to the context:

```tsx
// Small for lists
<Avatar size="sm" name="John Doe" />

// Medium for cards
<Avatar size="md" name="John Doe" />

// Large for profiles
<Avatar size="xl" name="John Doe" />
```

### 3. Handle Loading States

Provide feedback during image loading:

```tsx
<Avatar
  src="/user-avatar.jpg"
  name="John Doe"
  loading={isLoading}
  onLoad={() => setIsLoading(false)}
  onError={() => setIsLoading(false)}
/>
```
