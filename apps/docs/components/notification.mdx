# Notification

The Notification component provides a way to display important messages, alerts, and updates to users in a non-intrusive manner.

## Import

```tsx
import { Notification } from "@liquidify/components";
```

## Basic Usage

<CodeGroup>
```tsx React
import { Notification } from "@liquidify/components";

function App() {
return (

<Notification
  title="Success!"
  message="Your changes have been saved successfully."
  type="success"
/>
); }

````

```tsx Next.js
"use client";
import { Notification } from "@liquidify/components";

export default function Page() {
  return (
    <Notification
      title="Welcome!"
      message="Thanks for joining our platform."
      type="info"
    />
  );
}
````

</CodeGroup>

## With Actions

```tsx
<Notification
  title="Update Available"
  message="A new version is ready to install."
  type="info"
  actions={[
    { label: "Update Now", onClick: () => console.log("Updating...") },
    {
      label: "Later",
      variant: "ghost",
      onClick: () => console.log("Dismissed"),
    },
  ]}
/>
```

## Auto Dismiss

```tsx
<Notification
  title="Temporary Message"
  message="This will disappear automatically."
  type="info"
  autoHide={true}
  duration={5000} // 5 seconds
/>
```

## Props

| Prop        | Type                                                            | Default  | Description                    |
| ----------- | --------------------------------------------------------------- | -------- | ------------------------------ |
| `title`     | `string`                                                        | -        | Notification title             |
| `message`   | `string`                                                        | -        | Notification message           |
| `type`      | `"success" \| "warning" \| "error" \| "info"`                   | `"info"` | Notification type              |
| `autoHide`  | `boolean`                                                       | `false`  | Auto-hide after duration       |
| `duration`  | `number`                                                        | `5000`   | Duration before auto-hide (ms) |
| `actions`   | `Array<{label: string, onClick: () => void, variant?: string}>` | -        | Action buttons                 |
| `onClose`   | `() => void`                                                    | -        | Close callback                 |
| `className` | `string`                                                        | -        | Additional CSS classes         |

## Accessibility

- Uses ARIA live regions for screen reader announcements
- Keyboard navigation support for action buttons
- Focus management when notification appears
- Proper color contrast for all variants
