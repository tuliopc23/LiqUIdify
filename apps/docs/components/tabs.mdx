# Tabs

Accessible tabbed interface for switching between content panels.

{/* Live preview */}

<TabsExample />

## Installation

```bash
npm install liquidify
```

## Basic Usage

```tsx
import { Tabs } from "liquidify";

export default function Demo() {
  const tabs = [
    {
      id: "overview",
      label: "Overview",
      content: <div>Overview content here...</div>,
    },
    {
      id: "details",
      label: "Details",
      content: <div>Detailed information...</div>,
    },
    {
      id: "settings",
      label: "Settings",
      content: <div>Settings panel...</div>,
    },
  ];

  return <Tabs tabs={tabs} defaultTab="details" />;
}
```

## Props

| Prop         | Type        | Default | Description                 |
| ------------ | ----------- | ------- | --------------------------- |
| `tabs`       | `TabItem[]` | -       | Array of tab configurations |
| `defaultTab` | `string`    | -       | ID of initially active tab  |

## Tab Item Structure

```typescript
type TabItem = {
  id: string;
  label: string;
  content: ReactNode;
  disabled?: boolean;
};
```
