# GlassTabs

The `GlassTabs` component provides an elegant way to organize content into tabbed panels with beautiful glassmorphism effects and smooth animations.

## Import

```tsx
import { GlassTabs } from "@liquidify/components";
```

## Component API

### GlassTabs

| Prop             | Type                                  | Default        | Description           |
| ---------------- | ------------------------------------- | -------------- | --------------------- |
| `value`          | `string`                              | -              | Controlled active tab |
| `defaultValue`   | `string`                              | -              | Default active tab    |
| `onValueChange`  | `(value: string) => void`             | -              | Tab change handler    |
| `orientation`    | `'horizontal' \| 'vertical'`          | `'horizontal'` | Tab orientation       |
| `variant`        | `'default' \| 'pills' \| 'underline'` | `'default'`    | Visual variant        |
| `size`           | `'sm' \| 'md' \| 'lg'`                | `'md'`         | Tab size              |
| `activationMode` | `'automatic' \| 'manual'`             | `'automatic'`  | Activation behavior   |

### GlassTabs.List

| Prop        | Type      | Default | Description              |
| ----------- | --------- | ------- | ------------------------ |
| `className` | `string`  | -       | Additional CSS classes   |
| `loop`      | `boolean` | `true`  | Loop keyboard navigation |

### GlassTabs.Trigger

| Prop       | Type        | Default | Description |
| ---------- | ----------- | ------- | ----------- |
| `value`    | `string`    | -       | Tab value   |
| `disabled` | `boolean`   | `false` | Disable tab |
| `icon`     | `ReactNode` | -       | Tab icon    |
| `badge`    | `ReactNode` | -       | Tab badge   |

### GlassTabs.Content

| Prop        | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `value`     | `string` | -       | Tab value              |
| `className` | `string` | -       | Additional CSS classes |

### Controlled Tabs with Loading

```tsx
function ControlledTabsWithLoading() {
  const [activeTab, setActiveTab] = useState("data");
  const [loading, setLoading] = useState(false);

  const handleTabChange = async (value: string) => {
    setLoading(true);
    // Simulate API call
    await new Promise((resolve) => setTimeout(resolve, 1000));
    setActiveTab(value);
    setLoading(false);
  };

  return (
    <GlassTabs
      value={activeTab}
      onValueChange={handleTabChange}
      activationMode="manual"
    >
      <GlassTabs.List>
        <GlassTabs.Trigger value="data">Data</GlassTabs.Trigger>
        <GlassTabs.Trigger value="charts">Charts</GlassTabs.Trigger>
        <GlassTabs.Trigger value="export">Export</GlassTabs.Trigger>
      </GlassTabs.List>

      {loading ? (
        <div className="flex items-center justify-center p-8">
          <GlassSpinner size="lg" />
        </div>
      ) : (
        <>
          <GlassTabs.Content value="data">Data table content</GlassTabs.Content>
          <GlassTabs.Content value="charts">
            Chart visualizations
          </GlassTabs.Content>
          <GlassTabs.Content value="export">Export options</GlassTabs.Content>
        </>
      )}
    </GlassTabs>
  );
}
```

## Accessibility

The `GlassTabs` component follows WAI-ARIA guidelines:

- **Keyboard Navigation**: Arrow keys navigate between tabs, Tab moves to content
- **ARIA Attributes**: Proper `role="tablist"`, `role="tab"`, and `role="tabpanel"`
- **Focus Management**: Focus moves appropriately between tabs and content
- **Selection**: Space/Enter activates tabs in manual mode
