# Error Boundary

```tsx
<GlassErrorBoundary fallback={<div>Something went wrong.</div>}>
  <App />
</GlassErrorBoundary>
```

## Props

| Prop       | Type                                                   | Description                         |
| ---------- | ------------------------------------------------------ | ----------------------------------- |
| `fallback` | `ReactNode`                                            | UI rendered when an error is caught |
| `onError`  | `(error:Error, info: { componentStack:string })=>void` | Reporting hook                      |
