# Glass Grid

The Glass Grid component provides a flexible, responsive grid system with beautiful glass morphism effects. Perfect for creating modern layouts with a frosted glass aesthetic.

## Basic Usage

```tsx
import { GlassGrid, GlassGridItem } from "@liquidify/react";

function BasicGrid() {
  return (
    <GlassGrid columns={3} gap="md">
      <GlassGridItem>
        <div>Item 1</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Item 2</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Item 3</div>
      </GlassGridItem>
    </GlassGrid>
  );
}
```

## Responsive Grid

Create responsive layouts that adapt to different screen sizes:

```tsx
import { GlassGrid, GlassGridItem } from "@liquidify/react";

function ResponsiveGrid() {
  return (
    <GlassGrid
      columns={{
        xs: 1,
        sm: 2,
        md: 3,
        lg: 4,
        xl: 6,
      }}
      gap={{
        xs: "sm",
        md: "lg",
      }}
    >
      {Array.from({ length: 12 }, (_, i) => (
        <GlassGridItem key={i}>
          <div className="p-4">Item {i + 1}</div>
        </GlassGridItem>
      ))}
    </GlassGrid>
  );
}
```

## Grid Item Spanning

Control how grid items span across columns and rows:

```tsx
import { GlassGrid, GlassGridItem } from "@liquidify/react";

function SpanningGrid() {
  return (
    <GlassGrid columns={4} gap="md">
      <GlassGridItem colSpan={2}>
        <div>Spans 2 columns</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Item 2</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Item 3</div>
      </GlassGridItem>
      <GlassGridItem colSpan={3}>
        <div>Spans 3 columns</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Item 5</div>
      </GlassGridItem>
    </GlassGrid>
  );
}
```

## Auto-Fit Grid

Create grids that automatically fit items based on minimum width:

```tsx
import { GlassGrid, GlassGridItem } from "@liquidify/react";

function AutoFitGrid() {
  return (
    <GlassGrid autoFit minItemWidth="250px" gap="lg">
      {Array.from({ length: 8 }, (_, i) => (
        <GlassGridItem key={i}>
          <div className="p-6">
            <h3>Card {i + 1}</h3>
            <p>Auto-fitting grid item content</p>
          </div>
        </GlassGridItem>
      ))}
    </GlassGrid>
  );
}
```

## Glass Effects Customization

Customize the glass morphism effects:

```tsx
import { GlassGrid, GlassGridItem } from "@liquidify/react";

function CustomGlassGrid() {
  return (
    <GlassGrid
      columns={3}
      gap="md"
      glassEffect={{
        blur: 20,
        opacity: 0.15,
        borderOpacity: 0.3,
        shadow: "lg",
      }}
    >
      <GlassGridItem
        glassEffect={{
          blur: 15,
          opacity: 0.2,
          borderOpacity: 0.4,
        }}
      >
        <div>Custom glass item</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Default glass item</div>
      </GlassGridItem>
      <GlassGridItem>
        <div>Another item</div>
      </GlassGridItem>
    </GlassGrid>
  );
}
```

## Masonry Layout

Create a Pinterest-style masonry layout:

```tsx
import { GlassGrid, GlassGridItem } from "@liquidify/react";

function MasonryGrid() {
  const items = [
    { height: 200, content: "Short item" },
    { height: 300, content: "Medium item" },
    { height: 150, content: "Tiny item" },
    { height: 400, content: "Tall item" },
    { height: 250, content: "Another medium" },
    { height: 180, content: "Small item" },
  ];

  return (
    <GlassGrid masonry columns={3} gap="md">
      {items.map((item, i) => (
        <GlassGridItem key={i}>
          <div style={{ height: item.height }} className="p-4">
            {item.content}
          </div>
        </GlassGridItem>
      ))}
    </GlassGrid>
  );
}
```

## API Reference

### GlassGrid Props

| Prop           | Type                                            | Default   | Description                         |
| -------------- | ----------------------------------------------- | --------- | ----------------------------------- |
| `columns`      | `number \| ResponsiveValue<number>`             | `1`       | Number of columns in the grid       |
| `gap`          | `SpacingValue \| ResponsiveValue<SpacingValue>` | `'md'`    | Gap between grid items              |
| `autoFit`      | `boolean`                                       | `false`   | Enable auto-fit behavior            |
| `minItemWidth` | `string`                                        | `'250px'` | Minimum width for auto-fit items    |
| `masonry`      | `boolean`                                       | `false`   | Enable masonry layout               |
| `glassEffect`  | `GlassEffectProps`                              | -         | Glass morphism effect configuration |
| `className`    | `string`                                        | -         | Additional CSS classes              |
| `children`     | `ReactNode`                                     | -         | Grid items                          |

### GlassGridItem Props

| Prop          | Type                                | Default | Description                         |
| ------------- | ----------------------------------- | ------- | ----------------------------------- |
| `colSpan`     | `number \| ResponsiveValue<number>` | `1`     | Number of columns to span           |
| `rowSpan`     | `number \| ResponsiveValue<number>` | `1`     | Number of rows to span              |
| `glassEffect` | `GlassEffectProps`                  | -       | Glass morphism effect configuration |
| `className`   | `string`                            | -       | Additional CSS classes              |
| `children`    | `ReactNode`                         | -       | Item content                        |

### GlassEffectProps

| Prop            | Type                           | Default | Description             |
| --------------- | ------------------------------ | ------- | ----------------------- |
| `blur`          | `number`                       | `12`    | Backdrop blur intensity |
| `opacity`       | `number`                       | `0.1`   | Background opacity      |
| `borderOpacity` | `number`                       | `0.2`   | Border opacity          |
| `shadow`        | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'`  | Shadow size             |

### ResponsiveValue

```tsx
type ResponsiveValue<T> = {
  xs?: T;
  sm?: T;
  md?: T;
  lg?: T;
  xl?: T;
};
```

```tsx
function PhotoGallery() {
```
