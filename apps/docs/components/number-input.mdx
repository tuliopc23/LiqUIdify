## Props

| Prop       | Type               | Description    |
| ---------- | ------------------ | -------------- |
| `value`    | `number`           | Current value  |
| `onChange` | `(n:number)=>void` | Change handler |
| `min`      | `number`           | Min value      |
| `max`      | `number`           | Max value      |
| `step`     | `number`           | Increment step |
