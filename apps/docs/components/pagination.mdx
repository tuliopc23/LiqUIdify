# Pagination

The `GlassPagination` component provides elegant pagination controls with glassmorphism effects. It features smooth animations, intelligent ellipsis handling, and comprehensive accessibility support.

## Live Examples

### Basic Pagination

```tsx live
import { useState } from "react";
import { GlassPagination } from "liquidify";

export default function Demo() {
  const [currentPage, setCurrentPage] = useState(1);
  const totalPages = 10;

  return (
    <div
      style={{
        display: "flex",
        flexDirection: "column",
        gap: 16,
        alignItems: "center",
      }}
    >
      <p>
        Current Page: {currentPage} of {totalPages}
      </p>
      <GlassPagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
      />
    </div>
  );
}
```

### Configuration Options

```tsx live
import { useState } from "react";
import { GlassPagination, GlassButton } from "liquidify";

export default function Demo() {
  const [currentPage, setCurrentPage] = useState(5);
  const [totalPages, setTotalPages] = useState(20);
  const [showFirstLast, setShowFirstLast] = useState(true);
  const [showPrevNext, setShowPrevNext] = useState(true);
  const [showEllipsis, setShowEllipsis] = useState(true);
  const [siblingCount, setSiblingCount] = useState(1);

  return (
    <div
      style={{
        display: "flex",
        flexDirection: "column",
        gap: 20,
        alignItems: "center",
      }}
    >
      <div
        style={{
          display: "flex",
          gap: 12,
          flexWrap: "wrap",
          justifyContent: "center",
        }}
      >
        <GlassButton
          size="sm"
          variant={showFirstLast ? "primary" : "ghost"}
          onClick={() => setShowFirstLast(!showFirstLast)}
        >
          First/Last
        </GlassButton>
        <GlassButton
          size="sm"
          variant={showPrevNext ? "primary" : "ghost"}
          onClick={() => setShowPrevNext(!showPrevNext)}
        >
          Prev/Next
        </GlassButton>
        <GlassButton
          size="sm"
          variant={showEllipsis ? "primary" : "ghost"}
          onClick={() => setShowEllipsis(!showEllipsis)}
        >
          Ellipsis
        </GlassButton>
      </div>

      <div style={{ display: "flex", gap: 12, alignItems: "center" }}>
        <span>Siblings:</span>
        <GlassButton
          size="sm"
          onClick={() => setSiblingCount(Math.max(0, siblingCount - 1))}
        >
          -
        </GlassButton>
        <span>{siblingCount}</span>
        <GlassButton
          size="sm"
          onClick={() => setSiblingCount(siblingCount + 1)}
        >
          +
        </GlassButton>
      </div>

      <div style={{ display: "flex", gap: 12, alignItems: "center" }}>
        <span>Total Pages:</span>
        <GlassButton
          size="sm"
          onClick={() => setTotalPages(Math.max(1, totalPages - 5))}
        >
          -5
        </GlassButton>
        <span>{totalPages}</span>
        <GlassButton size="sm" onClick={() => setTotalPages(totalPages + 5)}>
          +5
        </GlassButton>
      </div>

      <GlassPagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
        showFirstLast={showFirstLast}
        showPrevNext={showPrevNext}
        showEllipsis={showEllipsis}
        siblingCount={siblingCount}
      />
    </div>
  );
}
```

### Large Dataset Example

```tsx live
import { useState } from "react";
import { GlassPagination } from "liquidify";

export default function Demo() {
  const [currentPage, setCurrentPage] = useState(50);
  const totalPages = 100;
  const itemsPerPage = 10;
  const totalItems = totalPages * itemsPerPage;

  const startItem = (currentPage - 1) * itemsPerPage + 1;
  const endItem = Math.min(currentPage * itemsPerPage, totalItems);

  return (
    <div
      style={{
        display: "flex",
        flexDirection: "column",
        gap: 16,
        alignItems: "center",
      }}
    >
      <div style={{ textAlign: "center" }}>
        <p>
          Showing {startItem}-{endItem} of {totalItems} items
        </p>
        <p style={{ fontSize: "0.875rem", opacity: 0.7 }}>
          Page {currentPage} of {totalPages}
        </p>
      </div>

      <GlassPagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
        siblingCount={2}
        boundaryCount={2}
      />
    </div>
  );
}
```

## Props

| Prop            | Type                              | Default        | Description                                 |
| --------------- | --------------------------------- | -------------- | ------------------------------------------- |
| `currentPage`   | `number`                          | -              | Current active page (1-indexed)             |
| `totalPages`    | `number`                          | -              | Total number of pages                       |
| `onPageChange`  | `(page: number) => void`          | -              | Callback when page changes                  |
| `showFirstLast` | `boolean`                         | `true`         | Show first/last page buttons                |
| `showPrevNext`  | `boolean`                         | `true`         | Show previous/next buttons                  |
| `showEllipsis`  | `boolean`                         | `true`         | Show ellipsis for truncated pages           |
| `siblingCount`  | `number`                          | `1`            | Number of pages to show around current page |
| `boundaryCount` | `number`                          | `1`            | Number of pages to show at start/end        |
| `disabled`      | `boolean`                         | `false`        | Disable all pagination controls             |
| `size`          | `"sm" \| "md" \| "lg"`            | `"md"`         | Size variant                                |
| `variant`       | `"default" \| "solid" \| "ghost"` | `"default"`    | Visual style variant                        |
| `className`     | `string`                          | `undefined`    | Additional CSS classes                      |
| `aria-label`    | `string`                          | `"Pagination"` | Accessibility label                         |

## Usage Patterns

### Table Pagination

```tsx
const TableWithPagination = ({ data }) => {
  const [currentPage, setCurrentPage] = useState(1);
  const itemsPerPage = 10;
  const totalPages = Math.ceil(data.length / itemsPerPage);

  const paginatedData = data.slice(
    (currentPage - 1) * itemsPerPage,
    currentPage * itemsPerPage,
  );

  return (
    <div>
      <Table data={paginatedData} />
      <GlassPagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
      />
    </div>
  );
};
```

### API Integration

```tsx
const ApiPagination = () => {
  const [currentPage, setCurrentPage] = useState(1);
  const [data, setData] = useState([]);
  const [totalPages, setTotalPages] = useState(0);

  useEffect(() => {
    fetchData(currentPage).then((response) => {
      setData(response.data);
      setTotalPages(response.totalPages);
    });
  }, [currentPage]);

  return (
    <div>
      <DataList data={data} />
      <GlassPagination
        currentPage={currentPage}
        totalPages={totalPages}
        onPageChange={setCurrentPage}
      />
    </div>
  );
};
```

### URL Synchronization

```tsx
const UrlSyncPagination = () => {
  const [searchParams, setSearchParams] = useSearchParams();
  const currentPage = parseInt(searchParams.get("page") || "1");

  const handlePageChange = (page) => {
    setSearchParams({ page: page.toString() });
  };

  return (
    <GlassPagination
      currentPage={currentPage}
      totalPages={totalPages}
      onPageChange={handlePageChange}
    />
  );
};
```

## Accessibility

### WCAG Compliance

- **WCAG 2.1 AA**: Full compliance with keyboard navigation and screen readers
- **Keyboard Navigation**: Arrow keys, Enter, and Space key support
- **Screen Readers**: Proper ARIA labels and current page announcements
- **Focus Management**: Clear focus indicators and logical tab order

### Implementation Details

```tsx
// Accessible pagination with proper ARIA
<GlassPagination
  currentPage={currentPage}
  totalPages={totalPages}
  onPageChange={setCurrentPage}
  aria-label="Product list pagination"
  role="navigation"
/>
```

### Best Practices

- Always provide meaningful `aria-label` for context
- Use `aria-current="page"` for the active page
- Ensure sufficient color contrast for all states
- Provide keyboard shortcuts for power users
- Include page information for screen readers
