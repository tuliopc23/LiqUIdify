# Dropdown

The `GlassDropdown` component provides beautiful glassmorphism dropdown menus with comprehensive keyboard navigation and accessibility features.

## Import

<CodeGroup>

    ```tsx Component Import
    import {GlassDropdown} from '@liquidify/components'
    ```

    ```tsx Bundle Import
    import {GlassDropdown} from '@liquidify/components/navigation'
    ```

    ```tsx All Imports
    import {
    GlassDropdown,
    GlassDropdownTrigger,
    GlassDropdownContent,
    GlassDropdownItem,
    GlassDropdownSeparator,
    GlassDropdownLabel
} from '@liquidify/components'
    ```

</CodeGroup>

## Props

### GlassDropdown

<ResponseField name="open" type="boolean">
    Controls the open state of the dropdown
</ResponseField>

<ResponseField name="onOpenChange" type="(open: boolean) => void">
    Callback when open state changes
</ResponseField>

<ResponseField name="modal" type="boolean" default="false">
    Whether to make the dropdown modal
</ResponseField>

### GlassDropdownTrigger

<ResponseField name="asChild" type="boolean" default="false">
    Render as child element instead of button
</ResponseField>

### GlassDropdownContent

<ResponseField name="align" type="'start' | 'center' | 'end'" default="center">
    Alignment relative to the trigger
</ResponseField>

<ResponseField name="side" type="'top' | 'right' | 'bottom' | 'left'" default="bottom">
    Which side to position the dropdown
</ResponseField>

<ResponseField name="sideOffset" type="number" default="4">
    Distance from the trigger
</ResponseField>

<ResponseField name="alignOffset" type="number" default="0">
    Offset along the alignment axis
</ResponseField>

<ResponseField name="avoidCollisions" type="boolean" default="true">
    Whether to avoid viewport collisions
</ResponseField>

<ResponseField name="loop" type="boolean" default="true">
    Whether keyboard navigation should loop
</ResponseField>

### GlassDropdownItem

<ResponseField name="disabled" type="boolean" default="false">
    Whether the item is disabled
</ResponseField>

<ResponseField name="shortcut" type="string">
    Keyboard shortcut to display
</ResponseField>

<ResponseField name="asChild" type="boolean" default="false">
    Render as child element
</ResponseField>

<ResponseField name="onSelect" type="(event: Event) => void">
    Callback when item is selected
</ResponseField>

## Accessibility

LiqUIdify dropdowns follow WAI-ARIA guidelines:

- **Keyboard Navigation**: Arrow keys, Enter, Escape, Tab support
- **Screen Reader Support**: Proper ARIA roles and attributes
- **Focus Management**: Automatic focus handling and restoration
- **Type-ahead**: Quick navigation by typing
- **Portal Rendering**: Avoids focus traps and z-index issues

### Accessibility Example
```tsx
function AccessibleDropdownExample() {
  const [isOpen, setIsOpen] = useState(false)

  return (
    <GlassDropdown open={isOpen} onOpenChange={setIsOpen}>
      <GlassDropdownTrigger
        aria-haspopup="true"
        aria-expanded={isOpen}
        aria-label="User account menu"
      >
        <User className="h-4 w-4" />
      </GlassDropdownTrigger>

      <GlassDropdownContent
        role="menu"
        aria-label="Account options"
      >
        <GlassDropdownItem role="menuitem">
          Profile
        </GlassDropdownItem>
        <GlassDropdownItem role="menuitem">
          Settings
        </GlassDropdownItem>
      </GlassDropdownContent>
    </GlassDropdown>
  )
}
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| **Space** | Open/close dropdown |
| **Enter** | Open/close dropdown or select item |
| **Arrow Down** | Move to next item |
| **Arrow Up** | Move to previous item |
| **Home** | Move to first item |
| **End** | Move to last item |
| **Escape** | Close dropdown |
| **Tab** | Close dropdown and move to next element |
| **A-Z** | Type-ahead search |

## Performance Notes

- Content is rendered in a portal for optimal stacking
- Items are virtualized for large lists (100+ items)
- Animations use GPU acceleration
- Keyboard navigation is debounced for smooth scrolling
- Type-ahead search is optimized for large option sets
