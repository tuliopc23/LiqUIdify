## Props

| Prop          | Type                                                   | Default     | Description                            |
| ------------- | ------------------------------------------------------ | ----------- | -------------------------------------- |
| `variant`     | `"primary" \| "secondary" \| "ghost" \| "destructive"` | `"primary"` | Button style variant                   |
| `size`        | `"sm" \| "md" \| "lg" \| "xl"`                         | `"md"`      | Button size                            |
| `leftIcon`    | `ReactNode`                                            | -           | Icon displayed on the left side        |
| `rightIcon`   | `ReactNode`                                            | -           | Icon displayed on the right side       |
| `loading`     | `boolean`                                              | `false`     | Show loading state with spinner        |
| `loadingText` | `string`                                               | -           | Text to show during loading state      |
| `fullWidth`   | `boolean`                                              | `false`     | Make button full width                 |
| `iconOnly`    | `boolean`                                              | `false`     | Icon-only button (square aspect ratio) |
| `disabled`    | `boolean`                                              | `false`     | Disabled state                         |
| `children`    | `ReactNode`                                            | -           | Button content                         |
| `className`   | `string`                                               | -           | Additional CSS classes                 |
| `onClick`     | `(event) => void`                                      | -           | Click event handler                    |

## Usage Patterns

### Form Actions

```tsx
function FormActions({ onSave, onCancel, isLoading }) {
  return (
    <div className="flex justify-end gap-3">
      <GlassButton variant="ghost" onClick={onCancel} disabled={isLoading}>
        Cancel
      </GlassButton>
      <GlassButton
        variant="primary"
        leftIcon={<Save className="w-4 h-4" />}
        loading={isLoading}
        loadingText="Saving..."
        onClick={onSave}
      >
        Save Changes
      </GlassButton>
    </div>
  );
}
```

## Accessibility

The GlassButton component follows WCAG 2.1 guidelines:

### Keyboard Navigation

- **Tab**: Navigate between buttons
- **Enter/Space**: Activate button
- **Escape**: Cancel action (context-dependent)

### Screen Reader Support

- Proper button semantics
- Loading state announcements
- Disabled state handling
- Icon descriptions when needed

## Best Practices

### Button Labels

- Use action verbs ("Save", "Delete", "Continue")
- Keep labels concise and descriptive
- Avoid generic terms like "Click here"
- Consider internationalization

### Visual Hierarchy

- Use primary buttons for main actions
- Limit primary buttons per view
- Group related actions together
- Maintain consistent spacing
