## Core Props

| Prop       | Type      | Description       |
| ---------- | --------- | ----------------- | ---------------- |
| `value`    | `Date     | null`             | Controlled value |
| `onChange` | `(d: Date | null) => void`    | Change handler   |
| `showTime` | `boolean` | Enable time field |
| `minDate`  | `Date`    | Lower bound       |
| `maxDate`  | `Date`    | Upper bound       |

## Tips

- Use `minDate` to block past dates.
- Provide a formatted summary near complex range pickers.
