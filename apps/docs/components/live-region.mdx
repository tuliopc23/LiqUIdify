Overview
Live Region helps announce updates (e.g., notifications) without moving focus.

Import
```ts
import { GlassLiveRegion } from '@liquidify/components'
```

Usage
```tsx
<GlassLiveRegion politeness="polite" atomic>
  {message}
</GlassLiveRegion>
```

Props
- politeness: 'off' | 'polite' | 'assertive' (default: 'polite')
- atomic?: boolean â€” read entire region on change

Accessibility
- Use assertive sparingly for critical alerts; prefer polite for non-urgent updates.
