# Migration Guide

This guide helps you migrate between versions of LiqUIdify and provides compatibility information for upgrading your applications.

## Latest Migration (v1.3.0 - Production Release)

### Key Changes

- Full TypeScript support with strict mode
- Enhanced animation system with GPU acceleration
- Bundle size optimization (Core: 28KB, Full: 58KB)
- New accessibility testing framework
- Complete documentation overhaul

### Installation

```bash
# Using npm
npm install @liquidify/components

# Using Bun (recommended)
bun add @liquidify/components
```

### Breaking Changes

1. Package name update

```diff
- import { Button } from "liquidify"
+ import { GlassButton } from "@liquidify/components"
```

2. Theme system redesign

```diff
- createTheme({ colors: { primary: '#000' } })
+ defineTheme({ colors: { primary: { 500: '#000' } } })
```

3. Animation hooks API

```diff
- useGlassEffect({ blur: 10 })
+ useGlassEffect({ blur: 10, performance: 'high' })
```

### Migration Steps

1. Update dependencies
2. Run automated migration tool
3. Update import statements
4. Review and update theme configuration
5. Test thoroughly

For detailed information about migrating from specific versions or other UI libraries, see our [complete migration guide](/guide/migration).
