# Command Palette

### Example

```tsx

const ITEMS = [
  { id: "new", label: "New File" },
  { id: "open", label: "Open..." },
  { id: "save", label: "Save" },
];

export default function Demo() {
  return <GlassCommand items={ITEMS} placeholder="Type a command" />;
}
```

## Props

| Prop          | Type                         | Description        |
| ------------- | ---------------------------- | ------------------ |
| `items`       | `{id:string;label:string}[]` | Command entries    |
| `onSelect`    | `(id:string)=>void`          | Selection callback |
| `placeholder` | `string`                     | Input placeholder  |
