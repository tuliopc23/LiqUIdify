# Glass Button

The `GlassButton` component brings the elegance of glassmorphism to interactive buttons, providing a smooth, modern experience with customizable glass effects.

## Import

<CodeGroup>

```tsx Component Import
import { GlassButton } from "@liquidify/components";
```

```tsx Bundle Import
import { GlassButton } from "@liquidify/components/core";
```

</CodeGroup>

## Basic Usage

<CodeGroup>

## Props

<ResponseField name="variant" type="string" default="default">
  The visual style of the button - `"default"` - Standard glass effect -
  `"primary"` - Emphasized primary action - `"secondary"` - Secondary action
  style - `"ghost"` - Minimal ghost appearance - `"destructive"` -
  Warning/danger style
</ResponseField>

<ResponseField name="size" type="string" default="md">
  The size of the button - `"xs"` - Extra small (24px height) - `"sm"` - Small
  (32px height) - `"md"` - Medium (40px height) - `"lg"` - Large (48px height) -
  `"xl"` - Extra large (56px height)
</ResponseField>

<ResponseField name="loading" type="boolean" default="false">
  Shows loading spinner and disables interaction
</ResponseField>

<ResponseField name="disabled" type="boolean" default="false">
  Disables the button and reduces opacity
</ResponseField>

<ResponseField name="fullWidth" type="boolean" default="false">
  Makes the button take the full width of its container
</ResponseField>

<ResponseField name="leftIcon" type="ReactNode">
  Icon to display on the left side of the button text
</ResponseField>

<ResponseField name="rightIcon" type="ReactNode">
  Icon to display on the right side of the button text
</ResponseField>

<ResponseField name="loadingText" type="string">
  Text to display when loading (overrides children)
</ResponseField>

<ResponseField name="magneticHover" type="boolean" default="false">
  Enables magnetic hover effect that follows cursor
</ResponseField>

<ResponseField name="rippleEffect" type="boolean" default="false">
  Enables ripple effect on click
</ResponseField>

<ResponseField name="glassEffect" type="object">
  Custom glass effect configuration
  ```tsx
  {
    intensity?: 'light' | 'medium' | 'strong' | 'ultra'
    blur?: boolean
    backdrop?: boolean
  }
  ```
</ResponseField>

<ResponseField name="asChild" type="boolean" default="false">
  When true, the component will render as its child element
</ResponseField>

## Events

<ResponseField name="onClick" type="(event: MouseEvent) => void">
  Called when button is clicked
</ResponseField>

<ResponseField name="onFocus" type="(event: FocusEvent) => void">
  Called when button receives focus
</ResponseField>

<ResponseField name="onBlur" type="(event: FocusEvent) => void">
  Called when button loses focus
</ResponseField>

## Accessibility

LiqUIdify buttons follow WAI-ARIA guidelines and include:

- **Keyboard Navigation**: Full support for Tab, Enter, and Space keys
- **Screen Reader Support**: Proper role, state, and property announcements
- **Focus Management**: Clear focus indicators that respect user preferences
- **Loading States**: Announced to screen readers with proper `aria-busy`
- **Disabled States**: Properly communicated via `aria-disabled`

### Accessibility Example

```tsx
<GlassButton
  variant="primary"
  disabled={!isValid}
  loading={isSubmitting}
  aria-label="Save document changes"
  aria-describedby="save-help"
  onClick={handleSave}
>
  {isSubmitting ? 'Saving...' : 'Save'}
</GlassButton>

<div id="save-help" className="sr-only">
  Saves all current changes to your document
</div>
```

## Theming

Customize button appearance using CSS variables:

```css
.custom-button {
  --glass-button-bg: rgba(59, 130, 246, 0.1);
  --glass-button-border: rgba(59, 130, 246, 0.3);
  --glass-button-text: rgb(59, 130, 246);
  --glass-button-shadow: 0 4px 16px rgba(59, 130, 246, 0.2);

  /* Hover states */
  --glass-button-bg-hover: rgba(59, 130, 246, 0.15);
  --glass-button-border-hover: rgba(59, 130, 246, 0.4);

  /* Focus states */
  --glass-button-ring: rgba(59, 130, 246, 0.5);
}
```

## Performance Notes

- Button components are highly optimized with `React.memo`
- Glass effects use GPU acceleration for smooth animations
- Icons should be properly sized (typically 16px for most sizes)
- Loading states automatically prevent double-submissions
- Ripple and magnetic effects are rendered efficiently with CSS transforms
