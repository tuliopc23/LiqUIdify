# Toast

Toast notification system with provider and hook for displaying temporary messages.

## Installation

```bash
npm install liquidify
```

## Basic Usage

```tsx
import { ToastProvider, useToast, GlassButton } from "liquidify";

function ToastDemo() {
  const { addToast } = useToast();
  
  return (
    <div style={{ display: "flex", gap: 12, flexWrap: "wrap" }}>
      <GlassButton
        onClick={() =>
          addToast({
            title: "Saved",
            description: "Changes persisted",
            type: "success",
          })
        }
      >
        Success
      </GlassButton>
      
      <GlassButton
        variant="ghost"
        onClick={() =>
          addToast({
            title: "Heads up",
            description: "Check your input",
            type: "warning",
          })
        }
      >
        Warning
      </GlassButton>
      
      <GlassButton
        variant="destructive"
        onClick={() =>
          addToast({
            title: "Error",
            description: "Something went wrong",
            type: "error",
          })
        }
      >
        Error
      </GlassButton>
    </div>
  );
}

export default function Demo() {
  return (
    <ToastProvider>
      <ToastDemo />
    </ToastProvider>
  );
}
```

## API

### ToastProvider

Wrapper component that provides toast context.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `position` | `ToastPosition` | `'top-right'` | Where toasts appear |
| `children` | `ReactNode` | - | App content |

### useToast Hook

Returns toast management functions:

```typescript
const { addToast, removeToast } = useToast();
```

#### addToast Options

| Option | Type | Description |
|--------|------|--------------|
| `title` | `string` | Toast title (optional) |
| `description` | `string` | Toast message |
| `type` | `'success' \| 'warning' \| 'error' \| 'info'` | Toast variant |
| `duration` | `number` | Auto-dismiss time in ms |
| `action` | `ReactNode` | Custom action element |
