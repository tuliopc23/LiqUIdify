# Glass Container

The `GlassContainer` component provides flexible layout containers with beautiful glassmorphism effects, perfect for organizing content sections and creating visual hierarchy.

## Import

<CodeGroup>

```tsx Component Import
import { GlassContainer } from "@liquidify/components";
```

```tsx Bundle Import
import { GlassContainer } from "@liquidify/components/layout";
```

</CodeGroup>

## Basic Usage

<CodeGroup>

```tsx Simple Container
<GlassContainer className="p-6">
  <h2>Welcome to LiqUIdify</h2>
  <p>Beautiful glassmorphism containers for your content.</p>
</GlassContainer>
```

```tsx Sectioned Layout
<div className="space-y-6">
  <GlassContainer variant="header" className="p-6">
    <h1>Header Section</h1>
  </GlassContainer>

  <GlassContainer variant="content" className="p-8">
    <p>Main content area with enhanced glass effects.</p>
  </GlassContainer>

  <GlassContainer variant="footer" className="p-4">
    <p>Footer content with subtle styling.</p>
  </GlassContainer>
</div>
```

</CodeGroup>

## Props

<ResponseField name="variant" type="string" default="default">
  The visual style of the container - `"default"` - Standard container -
  `"header"` - Optimized for headers and heroes - `"content"` - Enhanced for
  main content areas - `"sidebar"` - Styled for navigation and sidebars -
  `"footer"` - Designed for footer sections
</ResponseField>

<ResponseField name="size" type="string" default="md">
  The size of the container - `"xs"` - Extra small container - `"sm"` - Small
  container - `"md"` - Medium container - `"lg"` - Large container - `"xl"` -
  Extra large container - `"full"` - Full width/height
</ResponseField>

<ResponseField name="elevation" type="string" default="none">
  Shadow depth level - `"none"` - No shadow - `"sm"` - Small shadow - `"md"` -
  Medium shadow - `"lg"` - Large shadow - `"xl"` - Extra large shadow
</ResponseField>

<ResponseField name="bordered" type="boolean | string" default="false">
  Border styling - `false` - No border - `true` - Standard border - `"thick"` -
  Thick border - `"dashed"` - Dashed border
</ResponseField>

<ResponseField name="gradient" type="string">
  Background gradient preset - `"blue"` - Blue gradient - `"purple"` - Purple
  gradient - `"green"` - Green gradient - `"sunset"` - Orange-pink gradient -
  `"ocean"` - Blue-green gradient
</ResponseField>

<ResponseField name="glass" type="boolean" default="true">
  Enables glass effect styling
</ResponseField>

<ResponseField name="blurred" type="boolean" default="true">
  Enables backdrop blur effect
</ResponseField>

<ResponseField name="fullWidth" type="boolean" default="false">
  Makes container take full width
</ResponseField>

<ResponseField name="fullHeight" type="boolean" default="false">
  Makes container take full height
</ResponseField>

<ResponseField name="centered" type="boolean" default="false">
  Centers content within container
</ResponseField>

<ResponseField name="glassEffect" type="object">
  Custom glass effect configuration
  ```tsx
  {
    intensity?: 'light' | 'medium' | 'strong' | 'ultra';
    blur?: number;
    opacity?: number;
    saturation?: number;
  }
  ```
</ResponseField>

## Styling

### Default Classes

Glass containers come with default styling that can be extended:

```css
.glass-container {
  backdrop-filter: blur(16px) saturate(180%);
  background: rgba(255, 255, 255, 0.1);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 12px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.12);
}
```

### Custom Styling

```tsx
<GlassContainer
  className="custom-container"
  style={{
    "--glass-bg": "rgba(59, 130, 246, 0.1)",
    "--glass-border": "rgba(59, 130, 246, 0.3)",
    "--glass-blur": "blur(20px)",
  }}
>
  Custom styled container
</GlassContainer>
```

## Accessibility

- **Semantic HTML**: Uses appropriate container elements (`div`, `section`, `article`)
- **Landmark Roles**: Proper ARIA roles for different container types
- **Color Contrast**: Ensures sufficient contrast with glass effects
- **Keyboard Navigation**: Preserves focus flow through container content
- **Screen Reader Support**: Proper content structure and headings

### Accessibility Example

```tsx
<GlassContainer
  variant="content"
  role="main"
  aria-labelledby="main-heading"
  className="p-6"
>
  <h1 id="main-heading">Main Content</h1>
  <p>Accessible content within glass container</p>
</GlassContainer>
```

## Performance Notes

- Containers are optimized with `React.memo`
- Glass effects use efficient CSS properties
- Large layouts benefit from CSS Grid and Flexbox
- Backdrop filters are optimized for GPU acceleration
- Avoid excessive nesting of glass effects
