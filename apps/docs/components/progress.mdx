# Progress

The Progress component provides visual feedback about the completion status of a task or process.

## Import

```tsx
import { Progress } from "@liquidify/components";
```

## Basic Usage

<CodeGroup>
```tsx React
import { Progress } from "@liquidify/components";

function App() {
  return (
    <Progress value={65} max={100} />
  );
}
```

```tsx Next.js
"use client";
import { Progress } from "@liquidify/components";

export default function Page() {
  return (
    <Progress value={45} max={100} />
  );
}
```
</CodeGroup>

## With Label

```tsx
<Progress 
  value={75} 
  max={100} 
  label="Upload Progress"
  showValue={true}
/>
```

## Sizes

<CodeGroup>
```tsx Small
<Progress value={30} size="sm" />
```

```tsx Medium (Default)
<Progress value={50} size="md" />
```

```tsx Large
<Progress value={70} size="lg" />
```
</CodeGroup>

## Variants

<CodeGroup>


## Indeterminate

For unknown progress duration:

```tsx
<Progress indeterminate={true} />
```

## Circular Progress

```tsx
<Progress 
  value={60} 
  variant="circular" 
  size="lg"
  showValue={true}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | `0` | Current progress value |
| `max` | `number` | `100` | Maximum progress value |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Progress bar size |
| `variant` | `"default" \| "success" \| "warning" \| "error" \| "circular"` | `"default"` | Progress style variant |
| `label` | `string` | - | Accessible label |
| `showValue` | `boolean` | `false` | Show percentage value |
| `indeterminate` | `boolean` | `false` | Indeterminate progress |
| `className` | `string` | - | Additional CSS classes |

## Accessibility

- Uses `role="progressbar"` for screen readers
- Includes `aria-valuenow`, `aria-valuemin`, and `aria-valuemax` attributes
- Supports `aria-label` for descriptive text
- Keyboard accessible when interactive
