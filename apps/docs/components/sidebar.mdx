# Sidebar

The Sidebar component provides a flexible navigation sidebar for your application. It supports collapsible states, nested navigation, and responsive behavior.

## Basic Usage

```tsx
import { Sidebar, SidebarItem, SidebarGroup } from "@liquidify/react";

function BasicSidebar() {
  return (
    <Sidebar>
      <SidebarItem href="/dashboard" icon={<Home />}>
        Dashboard
      </SidebarItem>
      <SidebarItem href="/projects" icon={<Folder />}>
        Projects
      </SidebarItem>
      <SidebarItem href="/settings" icon={<Settings />}>
        Settings
      </SidebarItem>
    </Sidebar>
  );
}
```

## Collapsible Sidebar

Create a sidebar that can be collapsed to save space:

```tsx
import {
  Sidebar,
  SidebarItem,
  SidebarHeader,
  SidebarToggle,
} from "@liquidify/react";

function CollapsibleSidebar() {
  const [collapsed, setCollapsed] = useState(false);

  return (
    <Sidebar collapsed={collapsed} onCollapsedChange={setCollapsed}>
      <SidebarHeader>
        <div className="flex items-center gap-2">
          <img src="/logo.svg" alt="Logo" className="h-8 w-8" />
          {!collapsed && <span className="font-bold">LiqUIdify</span>}
        </div>
        <SidebarToggle />
      </SidebarHeader>

      <SidebarItem href="/dashboard" icon={<Home />}>
        Dashboard
      </SidebarItem>
      <SidebarItem href="/analytics" icon={<BarChart />}>
        Analytics
      </SidebarItem>
      <SidebarItem href="/users" icon={<Users />}>
        Users
      </SidebarItem>
    </Sidebar>
  );
}
```

## Grouped Navigation

Organize navigation items into logical groups:

```tsx
import { Sidebar, SidebarItem, SidebarGroup } from "@liquidify/react";

function GroupedSidebar() {
  return (
    <Sidebar>
      <SidebarGroup title="Main">
        <SidebarItem href="/dashboard" icon={<Home />}>
          Dashboard
        </SidebarItem>
        <SidebarItem href="/projects" icon={<Folder />}>
          Projects
        </SidebarItem>
      </SidebarGroup>

      <SidebarGroup title="Management">
        <SidebarItem href="/users" icon={<Users />}>
          Users
        </SidebarItem>
        <SidebarItem href="/roles" icon={<Shield />}>
          Roles
        </SidebarItem>
      </SidebarGroup>

      <SidebarGroup title="Settings">

          Billing
        </SidebarItem>
      </SidebarGroup>

}
```

## Nested Navigation

Create multi-level navigation with expandable sections:

```tsx
import { Sidebar, SidebarItem, SidebarSubmenu } from "@liquidify/react";

function NestedSidebar() {
  return (
    <Sidebar>
      <SidebarItem href="/dashboard" icon={<Home />}>
        Dashboard
      </SidebarItem>

      <SidebarSubmenu title="Projects" icon={<Folder />} defaultExpanded>
        <SidebarItem href="/projects/web">Web Projects</SidebarItem>
        <SidebarItem href="/projects/mobile">Mobile Apps</SidebarItem>
        <SidebarItem href="/projects/api">API Services</SidebarItem>
      </SidebarSubmenu>

      <SidebarSubmenu title="Analytics" icon={<BarChart />}>
        <SidebarItem href="/analytics/overview">Overview</SidebarItem>
        <SidebarItem href="/analytics/reports">Reports</SidebarItem>
        <SidebarItem href="/analytics/insights">Insights</SidebarItem>
      </SidebarSubmenu>

      <SidebarItem href="/settings" icon={<Settings />}>
        Settings
      </SidebarItem>
    </Sidebar>
  );
}
```

## With Footer

Add a footer section to the sidebar:

```tsx
import { Sidebar, SidebarItem, SidebarFooter } from "@liquidify/react";

function SidebarWithFooter() {
  const { user } = useAuth();

  return (
    <Sidebar>
      <SidebarItem href="/dashboard" icon={<Home />}>
        Dashboard
      </SidebarItem>
      <SidebarItem href="/projects" icon={<Folder />}>
        Projects
      </SidebarItem>
      <SidebarItem href="/team" icon={<Users />}>
        Team
      </SidebarItem>

      <SidebarFooter>
        <div className="flex items-center gap-3 p-3">
          <Avatar src={user.avatar} size="sm" />
          <div className="flex-1 min-w-0">
            <p className="text-sm font-medium truncate">{user.name}</p>
            <p className="text-xs text-gray-500 truncate">{user.email}</p>
          </div>
          <Button variant="ghost" size="sm" isIconOnly>
            <LogOut size={16} />
          </Button>
        </div>
      </SidebarFooter>
    </Sidebar>
  );
}
```

## Mobile Responsive

The sidebar adapts to mobile screens with overlay behavior:

```tsx
import { Sidebar, SidebarItem, useSidebar } from "@liquidify/react";

function ResponsiveSidebar() {
  const { isOpen, setIsOpen } = useSidebar();

  return (
    <>
      {/* Mobile overlay */}
      {isOpen && (
        <div
          className="fixed inset-0 bg-black/50 z-40 lg:hidden"
          onClick={() => setIsOpen(false)}
        />
      )}

      <Sidebar
        className={`
          fixed inset-y-0 left-0 z-50 w-64 transform transition-transform
          ${isOpen ? "translate-x-0" : "-translate-x-full"}
          lg:translate-x-0 lg:static lg:inset-0
        `}
      >
        <SidebarItem href="/dashboard" icon={<Home />}>
          Dashboard
        </SidebarItem>
        <SidebarItem href="/projects" icon={<Folder />}>
          Projects
        </SidebarItem>
        <SidebarItem href="/settings" icon={<Settings />}>
          Settings
        </SidebarItem>
      </Sidebar>
    </>
  );
}
```

## With Badges and Indicators

Add badges and status indicators to sidebar items:

```tsx
import { Sidebar, SidebarItem, Badge } from "@liquidify/react";

function SidebarWithBadges() {
  const { notifications, messages } = useNotifications();

  return (
    <Sidebar>
      <SidebarItem href="/dashboard" icon={<Home />}>
        Dashboard
      </SidebarItem>

      <SidebarItem
        href="/messages"
        icon={<MessageSquare />}
        badge={messages.unread > 0 ? messages.unread : undefined}
      >
        Messages
      </SidebarItem>

      <SidebarItem
        href="/notifications"
        icon={<Bell />}
        badge={notifications.unread > 0 ? notifications.unread : undefined}
        badgeColor="danger"
      >
        Notifications
      </SidebarItem>

      <SidebarItem href="/tasks" icon={<CheckSquare />} indicator="success">
        Tasks
      </SidebarItem>
    </Sidebar>
  );
}
```

## API Reference

### Sidebar Props

| Prop                  | Type                              | Default    | Description                 |
| --------------------- | --------------------------------- | ---------- | --------------------------- |
| `collapsed`           | `boolean`                         | `false`    | Collapsed state             |
| `collapsible`         | `boolean`                         | `false`    | Enable collapsible behavior |
| `width`               | `string \| number`                | `'16rem'`  | Sidebar width               |
| `collapsedWidth`      | `string \| number`                | `'4rem'`   | Width when collapsed        |
| `position`            | `'fixed' \| 'sticky' \| 'static'` | `'static'` | Positioning                 |
| `onCollapsedChange`   | `(collapsed: boolean) => void`    | -          | Collapse state handler      |
| `className`           | `string`                          | -          | Additional CSS classes      |
| `itemClassName`       | `string`                          | -          | Default item classes        |
| `activeItemClassName` | `string`                          | -          | Active item classes         |
| `children`            | `ReactNode`                       | -          | Sidebar content             |

### SidebarItem Props

| Prop         | Type                                                             | Default     | Description            |
| ------------ | ---------------------------------------------------------------- | ----------- | ---------------------- |
| `href`       | `string`                                                         | -           | Navigation link        |
| `icon`       | `ReactNode`                                                      | -           | Leading icon           |
| `badge`      | `string \| number`                                               | -           | Badge content          |
| `badgeColor` | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger'` | `'primary'` | Badge color            |
| `indicator`  | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger'` | -           | Status indicator       |
| `isActive`   | `boolean`                                                        | `false`     | Active state           |
| `isDisabled` | `boolean`                                                        | `false`     | Disabled state         |
| `onClick`    | `() => void`                                                     | -           | Click handler          |
| `className`  | `string`                                                         | -           | Additional CSS classes |
| `children`   | `ReactNode`                                                      | -           | Item content           |

### SidebarGroup Props

| Prop              | Type        | Default | Description              |
| ----------------- | ----------- | ------- | ------------------------ |
| `title`           | `string`    | -       | Group title              |
| `collapsible`     | `boolean`   | `false` | Enable collapsible group |
| `defaultExpanded` | `boolean`   | `true`  | Default expanded state   |
| `className`       | `string`    | -       | Additional CSS classes   |
| `children`        | `ReactNode` | -       | Group items              |

### SidebarSubmenu Props

| Prop              | Type               | Default | Description            |
| ----------------- | ------------------ | ------- | ---------------------- |
| `title`           | `string`           | -       | Submenu title          |
| `icon`            | `ReactNode`        | -       | Leading icon           |
| `defaultExpanded` | `boolean`          | `false` | Default expanded state |
| `badge`           | `string \| number` | -       | Badge content          |
| `className`       | `string`           | -       | Additional CSS classes |
| `children`        | `ReactNode`        | -       | Submenu items          |

## Accessibility

The Sidebar component includes accessibility features:

- Proper ARIA labels and roles
- Keyboard navigation support
- Focus management
- Screen reader compatibility

```tsx
<Sidebar aria-label="Main navigation">
  <SidebarItem href="/dashboard" aria-current={isActive ? "page" : undefined}>
    Dashboard
  </SidebarItem>
</Sidebar>
```

## Examples

### Admin Dashboard Sidebar

```tsx
function AdminSidebar() {
  const { user, permissions } = useAuth();

  return (
    <Sidebar collapsible>
      <SidebarHeader>
        <div className="flex items-center gap-2">
          <img src="/admin-logo.svg" alt="Admin" className="h-8 w-8" />
          <span className="font-bold">Admin Panel</span>
        </div>
      </SidebarHeader>

      <SidebarGroup title="Overview">
        <SidebarItem href="/admin/dashboard" icon={<Home />}>
          Dashboard
        </SidebarItem>
        <SidebarItem href="/admin/analytics" icon={<BarChart />}>
          Analytics
        </SidebarItem>
      </SidebarGroup>

      {permissions.includes("manage_users") && (
        <SidebarGroup title="User Management">
          <SidebarItem href="/admin/users" icon={<Users />}>
            Users
          </SidebarItem>
          <SidebarItem href="/admin/roles" icon={<Shield />}>
            Roles
          </SidebarItem>
        </SidebarGroup>
      )}

      <SidebarGroup title="Content">
        <SidebarItem href="/admin/posts" icon={<FileText />}>
          Posts
        </SidebarItem>
        <SidebarItem href="/admin/media" icon={<Image />}>
          Media
        </SidebarItem>
      </SidebarGroup>

      <SidebarFooter>
        <UserProfile user={user} />
      </SidebarFooter>
    </Sidebar>
  );
}
```
