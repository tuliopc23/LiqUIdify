---
title: Theme Provider
description: Provide app-wide theme context and CSS variables for Liquid Glass.
---

Overview
Theme Provider sets up theming tokens and context for all components.

Import
```ts
import { ThemeProvider } from '@liquidify/components'
```

Usage
```tsx
import { ThemeProvider } from '@liquidify/components'

export default function App() {
  return (
    <ThemeProvider defaultTheme="light" storageKey="liquid-theme">
      <YourApp />
    </ThemeProvider>
  )
}
```

Props
- defaultTheme?: 'light' | 'dark' | 'system' (default: 'system')
- storageKey?: string (default: 'theme')
- onThemeChange?: (theme: string) => void

Notes
- This provider sets CSS custom properties and data-theme attributes.
- Works with Theme Toggle for user-controlled switching.
