## Props

### Drawer (Root)

| Prop           | Type                      | Default | Description                      |
| -------------- | ------------------------- | ------- | -------------------------------- |
| `open`         | `boolean`                 | `false` | Whether the drawer is open       |
| `onOpenChange` | `(open: boolean) => void` | -       | Callback when open state changes |
| `children`     | `ReactNode`               | -       | Drawer content                   |

### Drawer.Content

| Prop                  | Type                                     | Default    | Description                       |
| --------------------- | ---------------------------------------- | ---------- | --------------------------------- |
| `side`                | `'top' \| 'bottom' \| 'left' \| 'right'` | `'right'`  | Which side the drawer slides from |
| `size`                | `'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'` | `'md'`     | Drawer size                       |
| `showCloseButton`     | `boolean`                                | `true`     | Show close button                 |
| `closeButtonPosition` | `'header' \| 'overlay'`                  | `'header'` | Close button position             |
| `className`           | `string`                                 | -          | Additional CSS classes            |
| `children`            | `ReactNode`                              | -          | Drawer content                    |

### Drawer.Header, Drawer.Body, Drawer.Footer

| Prop        | Type        | Description            |
| ----------- | ----------- | ---------------------- |
| `className` | `string`    | Additional CSS classes |
| `children`  | `ReactNode` | Section content        |

### Drawer.Title, Drawer.Description

| Prop        | Type        | Description            |
| ----------- | ----------- | ---------------------- |
| `className` | `string`    | Additional CSS classes |
| `children`  | `ReactNode` | Text content           |

## Usage Patterns

### Navigation Drawer

```tsx
function NavigationDrawer() {
  const [open, setOpen] = useState(false);

  const navigationItems = [
    { icon: Home, label: "Dashboard", href: "/dashboard" },
    { icon: User, label: "Profile", href: "/profile" },
    { icon: Settings, label: "Settings", href: "/settings" },
  ];

  return (
    <Drawer open={open} onOpenChange={setOpen}>
      <Drawer.Trigger asChild>
        <GlassButton variant="ghost" size="sm">
          <Menu className="h-4 w-4" />
        </GlassButton>
      </Drawer.Trigger>

      <Drawer.Content side="left" size="sm">
        <Drawer.Header>
          <Drawer.Title>Navigation</Drawer.Title>
        </Drawer.Header>

        <Drawer.Body>
          <nav className="space-y-2">
            {navigationItems.map((item) => (
              <a
                key={item.href}
                href={item.href}
                className="flex items-center gap-3 p-3 rounded-lg hover:bg-white/5 transition-colors"
                onClick={() => setOpen(false)}
              >
                <item.icon className="h-4 w-4" />
                <span>{item.label}</span>
              </a>
            ))}
          </nav>
        </Drawer.Body>

        <Drawer.Footer>
          <GlassButton variant="ghost" className="w-full justify-start">
            <LogOut className="h-4 w-4 mr-2" />
            Sign Out
          </GlassButton>
        </Drawer.Footer>
      </Drawer.Content>
    </Drawer>
  );
}
```

### Form Drawer

```tsx
function EditUserDrawer({ user, open, onOpenChange }) {
  const [formData, setFormData] = useState({
    name: user?.name || "",
    email: user?.email || "",
    role: user?.role || "",
  });

  const handleSubmit = (e) => {
    e.preventDefault();
    // Handle form submission
    onOpenChange(false);
  };

  return (
    <Drawer open={open} onOpenChange={onOpenChange}>
      <Drawer.Content side="right" size="md">
        <Drawer.Header>
          <Drawer.Title>Edit User</Drawer.Title>
          <Drawer.Description>
            Update user information and permissions.
          </Drawer.Description>
        </Drawer.Header>

        <Drawer.Body>
          <form onSubmit={handleSubmit} className="space-y-6">
            <GlassFormField label="Full Name" required>
              <GlassInput
                value={formData.name}
                onChange={(e) =>
                  setFormData((prev) => ({ ...prev, name: e.target.value }))
                }
                placeholder="Enter full name"
              />
            </GlassFormField>

            <GlassFormField label="Email Address" required>
              <GlassInput
                type="email"
                value={formData.email}
                onChange={(e) =>
                  setFormData((prev) => ({ ...prev, email: e.target.value }))
                }
                placeholder="Enter email address"
              />
            </GlassFormField>

            <GlassFormField label="Role">
              <GlassSelect
                options={[
                  { value: "admin", label: "Administrator" },
                  { value: "editor", label: "Editor" },
                  { value: "viewer", label: "Viewer" },
                ]}
                value={formData.role}
                onChange={(value) =>
                  setFormData((prev) => ({ ...prev, role: value }))
                }
                placeholder="Select role"
              />
            </GlassFormField>
          </form>
        </Drawer.Body>

        <Drawer.Footer>
          <GlassButton type="submit" variant="primary">
            Save Changes
          </GlassButton>
          <Drawer.Close asChild>
            <GlassButton variant="ghost">Cancel</GlassButton>
          </Drawer.Close>
        </Drawer.Footer>
      </Drawer.Content>
    </Drawer>
  );
}
```

## Accessibility

### WCAG Compliance

- **Focus Management**: Focus is trapped within the drawer when open
- **Keyboard Navigation**: Full keyboard support with Escape to close
- **Screen Reader**: Proper ARIA attributes and announcements
- **Focus Return**: Focus returns to trigger element when closed

### Keyboard Shortcuts

| Key           | Action                              |
| ------------- | ----------------------------------- |
| `Escape`      | Close the drawer                    |
| `Tab`         | Navigate through focusable elements |
| `Shift + Tab` | Navigate backwards through elements |

## Apple HIG Guidelines

### Visual Design

- **Clarity**: Clear visual hierarchy with proper spacing
- **Depth**: Layered glass effects that create spatial relationships
- **Consistency**: Consistent behavior across different sides and sizes
- **Feedback**: Smooth slide animations with proper easing

### Interaction Design

- **Predictable**: Familiar slide-out behavior
- **Responsive**: Immediate response to user interactions
- **Contextual**: Appropriate size and position for content
- **Dismissible**: Multiple ways to close (button, backdrop, escape)

### Content Organization

- **Structured**: Clear header, body, and footer sections
- **Scannable**: Easy to scan and navigate content
- **Focused**: Single purpose per drawer
- **Progressive**: Reveal information progressively

## Best Practices

### Do's

- ✅ Use appropriate sizes for content complexity
- ✅ Provide clear titles and descriptions
- ✅ Include close buttons for accessibility
- ✅ Handle focus management properly
- ✅ Use consistent animation timing

### Don'ts

- ❌ Don't nest drawers within drawers
- ❌ Don't make drawers too small for touch targets
- ❌ Don't forget to handle escape key
- ❌ Don't use drawers for critical actions without confirmation
- ❌ Don't make content too wide for mobile devices

### Performance

- **Lazy Loading**: Load drawer content only when needed
- **Animation Optimization**: Use transform for smooth animations
- **Portal Rendering**: Render outside main DOM tree
- **Memory Management**: Clean up event listeners on unmount

```tsx
// Lazy loading example
const LazyDrawerContent = lazy(() => import("./DrawerContent"));

function OptimizedDrawer({ open, onOpenChange }) {
  return (
    <Drawer open={open} onOpenChange={onOpenChange}>
      <Drawer.Content>
        {open && (
          <Suspense fallback={<div>Loading...</div>}>
            <LazyDrawerContent />
          </Suspense>
        )}
      </Drawer.Content>
    </Drawer>
  );
}
```
