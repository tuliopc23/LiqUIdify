## Props

| Prop       | Type                      | Description               |
| ---------- | ------------------------- | ------------------------- |
| `accept`   | `string`                  | Accepted MIME types       |
| `multiple` | `boolean`                 | Allow multi-select        |
| `maxSize`  | `number`                  | Optional size limit bytes |
| `onFiles`  | `(files: File[]) => void` | Callback after selection  |

## Accessibility

- Native input retained (visually hidden) ensuring keyboard & screen reader support.
