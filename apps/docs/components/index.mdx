# Components Overview

Below are representative production components and the core Liquid Glass building blocks. Examples are live MDX/React so you can interact with state and see behavior.

## Glass Button (live)

```tsx live
import { useState } from 'react'
import { GlassButton } from '@liquidify/components'

export default function Example() {
  const [count, setCount] = useState(0)
  return (
    <div style={{display:'flex', gap:12, alignItems:'center'}}>
      <GlassButton variant="primary" onClick={() => setCount(c => c + 1)}>
        Clicked {count}x
      </GlassButton>
      <GlassButton variant="ghost" onClick={() => setCount(0)}>
        Reset
      </GlassButton>
    </div>
  )
}
```

## Glass Card + Content (live)

```tsx live
import { GlassButton } from '@liquidify/components'

function GlassContainer({ children }) {
  return (
    <div className="liquid-glass-container liquid-glass-enhanced" style={{maxWidth:560}}>
      <div className="liquid-glass-filter" />
      <div className="liquid-glass-overlay" />
      <div className="liquid-glass-specular" />
      <div className="liquid-glass-content">
        {children}
      </div>
    </div>
  )
}

export default function Example() {
  return (
    <GlassContainer>
      <span>Liquid Glass container built from layered CSS.</span>
      <GlassButton variant="secondary">Action</GlassButton>
    </GlassContainer>
  )
}
```
