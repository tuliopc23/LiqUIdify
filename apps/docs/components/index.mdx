## Overview

LiqUIdify provides a comprehensive set of Apple-inspired components built with the liquid glass design system. All components feature accessibility-first design, smooth animations, and seamless dark mode support.

## Quick Start

### Installation

```bash
npm install liquidify
## or
bun add liquidify
```

### Basic Setup

```tsx
import { GlassUIProvider, ThemeProvider } from "liquidify";
import "liquidify/styles";

function App() {
  return (
    <ThemeProvider defaultTheme="light">
      <GlassUIProvider>{/* Your app content */}</GlassUIProvider>
    </ThemeProvider>
  );
}
```

## Live Examples

### Interactive Button Demo

```tsx live
import { useState } from "react";
import { GlassButton, ThemeToggle } from "liquidify";

export default function Example() {
  const [count, setCount] = useState(0);
  return (
    <div className="flex items-center gap-4 p-4">
      <GlassButton variant="primary" onClick={() => setCount((c) => c + 1)}>
        Clicked {count}x
      </GlassButton>
      <GlassButton variant="ghost" onClick={() => setCount(0)}>
        Reset
      </GlassButton>
      <ThemeToggle />
    </div>
  );
}
```

### Liquid Glass Container

```tsx live
import { LiquidGlassProvider, GlassButton } from "liquidify";

function GlassContainer({ children }) {
  return (
    <div className="liquid-glass-container liquid-glass-enhanced liquid-glass-md max-w-md">
      <div className="liquid-glass-filter" />
      <div className="liquid-glass-overlay" />
      <div className="liquid-glass-specular" />
      <div className="liquid-glass-content flex-col items-start">
        {children}
      </div>
    </div>
  );
}

export default function Example() {
  return (
    <LiquidGlassProvider>
      <GlassContainer>
        <h3 className="text-liquid-primary font-semibold mb-2">Liquid Glass</h3>
        <p className="text-liquid-secondary mb-4">
          Layered transparency with specular highlights and content-aware
          adaptation.
        </p>
        <GlassButton variant="secondary">Learn More</GlassButton>
      </GlassContainer>
    </LiquidGlassProvider>
  );
}
```

## Getting Started Guide

### 1. Install Dependencies

```bash
npm install liquidify lucide-react
```

### 2. Setup Providers

```tsx
import { GlassUIProvider, ThemeProvider } from "liquidify";
import "liquidify/styles";

function App() {
  return (
    <ThemeProvider defaultTheme="light" storageKey="my-app-theme">
      <GlassUIProvider
        glassConfig={{
          blur: 24,
          opacity: 0.75,
          adaptToContent: true,
        }}
      >
        <div className="min-h-screen bg-glass-bg-canvas">
          {/* Your app content */}
        </div>
      </GlassUIProvider>
    </ThemeProvider>
  );
}
```

### 3. Use Components

```tsx
import {
  GlassButton,
  GlassCard,
  GlassInput,
  ThemeToggle,
} from "liquidify";

function MyComponent() {
  return (
    <GlassCard className="max-w-md">
      <GlassCard.Header>
        <GlassCard.Title>Welcome</GlassCard.Title>
        <GlassCard.Description>
          Get started with LiqUIdify components
        </GlassCard.Description>
      </GlassCard.Header>
      <GlassCard.Content>
        <GlassInput placeholder="Enter your name" />
      </GlassCard.Content>
      <GlassCard.Actions>
        <GlassButton variant="primary">Get Started</GlassButton>
        <ThemeToggle />
      </GlassCard.Actions>
    </GlassCard>
  );
}
```

### 4. Customize Themes

```css
/* Custom theme variables */
:root {
  --lg-accent: #007aff; /* Apple blue */
  --lg-radius-md: 1.5rem; /* Larger radius */
  --lg-transition: all 0.3s ease; /* Custom timing */
}
```

## Design Principles

### Apple Human Interface Guidelines

All components follow Apple's design principles:

- **Clarity**: Clear visual hierarchy and legible typography
- **Deference**: Content takes precedence over interface
- **Depth**: Realistic layering creates spatial relationships

### Accessibility First

- WCAG 2.1 AA compliance
- Keyboard navigation support
- Screen reader compatibility
- Reduced motion preferences
- High contrast mode support

### Performance Optimized

- GPU-accelerated animations
- Efficient CSS custom properties
- Tree-shakeable imports
- Minimal runtime overhead
- SSR-compatible
