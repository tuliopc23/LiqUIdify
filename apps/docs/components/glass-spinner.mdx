# Glass Spinner

The Glass Spinner component provides an elegant loading indicator with glass morphism effects. Perfect for showing loading states while maintaining the modern glass aesthetic.

## Basic Usage

```tsx
import { GlassSpinner } from '@liquidify/react';

function BasicSpinner() {
  return <GlassSpinner />;
}
```

## Sizes

Available in multiple sizes to fit different contexts:

```tsx
import { GlassSpinner } from '@liquidify/react';

function SpinnerSizes() {
  return (
    <div className="flex items-center gap-4">
      <GlassSpinner size="xs" />
      <GlassSpinner size="sm" />
      <GlassSpinner size="md" />
      <GlassSpinner size="lg" />
      <GlassSpinner size="xl" />
    </div>
  );
}
```

## Loading States

Use with different loading scenarios:

```tsx
import { GlassSpinner, GlassButton } from '@liquidify/react';
import { useState } from 'react';

function LoadingButton() {
  const [loading, setLoading] = useState(false);

  const handleClick = async () => {
    setLoading(true);
    // Simulate API call
    await new Promise(resolve => setTimeout(resolve, 2000));
    setLoading(false);
  };

  return (
    <GlassButton onClick={handleClick} disabled={loading}>
      {loading ? (
        <>
          <GlassSpinner size="sm" />
          Loading...
        </>
      ) : (
        'Click me'
      )}
    </GlassButton>
  );
}
```

## Full Page Loading

Create a full-page loading overlay:

```tsx
import { GlassSpinner } from '@liquidify/react';

function FullPageLoader({ isLoading }: { isLoading: boolean }) {
  if (!isLoading) return null;

  return (
    <div className="fixed inset-0 bg-black/20 backdrop-blur-sm flex items-center justify-center z-50">
      <div className="flex flex-col items-center gap-4">
        <GlassSpinner size="xl" />
        <p className="text-white text-lg">Loading your content...</p>
      </div>
    </div>
  );
}
```

## Inline Loading

Show loading state inline with content:

```tsx
import { GlassSpinner } from '@liquidify/react';

function InlineLoader({ loading, children }: { loading: boolean; children: React.ReactNode }) {
  return (
    <div className="relative">
      {loading && (
        <div className="absolute inset-0 bg-white/50 backdrop-blur-sm flex items-center justify-center">
          <GlassSpinner />
        </div>
      )}
      <div className={loading ? 'opacity-50' : ''}>
        {children}
      </div>
    </div>
  );
}
```

## API Reference

### GlassSpinner Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Size of the spinner |
| `color` | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'error' \| string` | `'primary'` | Color of the spinner |
| `variant` | `'spin' \| 'pulse' \| 'bounce' \| 'dots'` | `'spin'` | Animation variant |
| `speed` | `'slow' \| 'normal' \| 'fast'` | `'normal'` | Animation speed |
| `glassEffect` | `GlassEffectProps` | - | Glass morphism effect configuration |
| `className` | `string` | - | Additional CSS classes |
| `aria-label` | `string` | `'Loading'` | Accessibility label |

### GlassEffectProps

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `blur` | `number` | `12` | Backdrop blur intensity |
| `opacity` | `number` | `0.1` | Background opacity |
| `borderOpacity` | `number` | `0.2` | Border opacity |
| `shadow` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Shadow size |

## Accessibility

The Glass Spinner component includes accessibility features:

- Proper ARIA labels for screen readers
- Respects `prefers-reduced-motion` settings
- Semantic loading indicators
- Keyboard navigation support

```tsx
// Reduced motion support
<GlassSpinner 
  variant="pulse" // Falls back to pulse for reduced motion
  aria-label="Loading content"
/>
```

## Best Practices

### 1. Provide Context

Always provide context about what's loading:

```tsx
// Good
<div>
  <GlassSpinner />
  <span className="sr-only">Loading user profile</span>
</div>

// Better
<div aria-live="polite">
  <GlassSpinner aria-label="Loading user profile" />
  <span>Loading user profile...</span>
</div>
```

### 2. Use Appropriate Sizes

Match spinner size to the context:

```tsx
// Small for buttons
<GlassButton disabled>
  <GlassSpinner size="xs" />
  Save
</GlassButton>

// Large for page loading
<div className="min-h-screen flex items-center justify-center">
  <GlassSpinner size="xl" />
</div>
```

### 3. Handle Loading States

Properly manage loading states:

```tsx
function DataComponent() {
  const { data, loading, error } = useQuery();

  if (loading) return <GlassSpinner />;
  if (error) return <ErrorMessage />;
  return <DataDisplay data={data} />;
}
```
