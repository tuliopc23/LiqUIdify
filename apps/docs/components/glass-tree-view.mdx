---
title: Glass Tree View
description: Hierarchical tree component with Liquid Glass styling.
---

Overview
Glass Tree View renders nested items that can be expanded and collapsed.

Import
```ts
import { GlassTreeView } from '@liquidify/components'
```

Usage
```tsx
const data = [
  { id: '1', label: 'Root', children: [
    { id: '1-1', label: 'Child' }
  ]}
]

<GlassTreeView items={data} />
```

Props
- items: { id: string; label: string; children?: Item[] }[]
- defaultExpandedIds?: string[]
- onToggle?: (id: string, expanded: boolean) => void
