# Loading

The Loading component provides visual feedback during asynchronous operations and data fetching.

## Import

```tsx
import { Loading } from "@liquidify/components";
```

## Basic Usage

<CodeGroup>
```tsx React
import { Loading } from "@liquidify/components";

function App() {
  return <Loading />;
}
```

```tsx Next.js
"use client";
import { Loading } from "@liquidify/components";

export default function Page() {
  return <Loading />;
}
```
</CodeGroup>

## With Text

```tsx
<Loading text="Loading..." />
```

## Sizes

<CodeGroup>
```tsx Small
<Loading size="sm" />
```

```tsx Medium (Default)
<Loading size="md" />
```

```tsx Large
<Loading size="lg" />
```
</CodeGroup>

## Variants

<CodeGroup>


```tsx Bars
<Loading variant="bars" />
```
</CodeGroup>

## Full Screen

```tsx
<Loading 
  fullScreen={true}
  text="Loading application..."
  backdrop={true}
/>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Loading indicator size |
| `variant` | `"spinner" \| "dots" \| "pulse" \| "bars"` | `"spinner"` | Loading animation type |
| `text` | `string` | - | Loading text |
| `fullScreen` | `boolean` | `false` | Full screen overlay |
| `backdrop` | `boolean` | `false` | Show backdrop |
| `color` | `string` | - | Custom color |
| `className` | `string` | - | Additional CSS classes |

## Accessibility

- Uses `role="status"` for screen reader announcements
- Includes `aria-label` for descriptive text
- Live region updates for dynamic loading states
