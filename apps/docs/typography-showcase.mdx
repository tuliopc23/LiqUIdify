## üìù Font Families

### Display Fonts

```css
font-family:
  "SF Pro Display",
  "Inter",
  -apple-system,
  system-ui;
```

Used for **headlines, titles, and large text** where impact and visual hierarchy are important.

### Text Fonts

```css
font-family:
  "SF Pro Text",
  "Inter",
  -apple-system,
  system-ui;
```

Optimized for **body text, paragraphs, and UI elements** with superior readability at smaller sizes.

### Monospace Fonts

```css
font-family: "SF Mono", "Fira Code", "JetBrains Mono", monospace;
```

Perfect for **code blocks, technical content,** and data that requires fixed-width alignment.

## üìê Line Heights

Optimized for different content types:

```css
--leading-tight: 1.25; /* Large headings */
--leading-snug: 1.375; /* Medium headings */
--leading-normal: 1.5; /* UI text */
--leading-relaxed: 1.625; /* Body paragraphs */
--leading-loose: 2; /* Spaced content */
```
