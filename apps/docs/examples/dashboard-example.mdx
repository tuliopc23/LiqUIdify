# Dashboard Example

This example demonstrates a typical dashboard layout built with LiqUIdify components. It includes a sidebar, navbar, cards, table, and modals.

## Layout Structure

```tsx
;
import "liquidify/styles";

export default function Dashboard() {
  const [open, setOpen] = React.useState(false);

  return (
    <div className="min-h-screen bg-liquid-readable">
      <Sidebar />
      <div className="ml-64">
        <Navbar title="Dashboard" />
        <main className="p-6 space-y-6">
          <div className="grid gap-6 sm:grid-cols-2 lg:grid-cols-3">
            <GlassCard variant="elevated" className="p-6">
              <GlassCard.Title>Total Users</GlassCard.Title>
              <div className="text-3xl font-bold">12,482</div>
            </GlassCard>
            <GlassCard variant="elevated" className="p-6">
              <GlassCard.Title>Revenue</GlassCard.Title>
              <div className="text-3xl font-bold">$78,320</div>
            </GlassCard>
            <GlassCard variant="elevated" className="p-6">
              <GlassCard.Title>Conversion</GlassCard.Title>
              <div className="text-3xl font-bold">3.2%</div>
            </GlassCard>
          </div>

          <GlassCard variant="flat" className="p-0">
            <div className="flex items-center justify-between p-4">
              <h3 className="text-lg font-semibold">Recent Activity</h3>
              <GlassButton onClick={() => setOpen(true)}>Add</GlassButton>
            </div>
            <GlassTable
              columns={[
                { key: "user", header: "User" },
                { key: "action", header: "Action" },
                { key: "date", header: "Date" },
              ]}
              data={[]}
            />
          </GlassCard>

          <GlassModal open={open} onOpenChange={setOpen}>
            <div className="p-6 space-y-4">
              <h3 className="text-lg font-semibold">Add Item</h3>
              <p>Modal content goes here.</p>
              <div className="flex justify-end gap-2">
                <GlassButton variant="ghost" onClick={() => setOpen(false)}>
                  Cancel
                </GlassButton>
                <GlassButton onClick={() => setOpen(false)}>Save</GlassButton>
              </div>
            </div>
          </GlassModal>
        </main>
      </div>
    </div>
  );
}
```

## Notes

- Grid and spacing use Tailwind utility classes
- Interactive patterns use GlassModal and buttons with accessible defaults
- Table is configured with typed columns and data
