## <span className="sf-icon-text"><span className="sf-icon sf-magnifyingglass"></span>Quick Start</span>

<Tip>
  Get started with Liquidify in seconds using your favorite package manager.
</Tip>

```bash
npm install @liquidify/react
# or
yarn add @liquidify/react
# or
pnpm add @liquidify/react
```

<Frame>
```tsx Basic Usage
import { GlassButton, ThemeProvider } from '@liquidify/react'

function App() {
  return (
    <ThemeProvider>
      <GlassButton variant="primary">Get Started</GlassButton>
    </ThemeProvider>
  )
}
```
</Frame>
