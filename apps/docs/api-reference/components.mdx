# Components API

Complete reference for all LiqUIdify glassmorphism components.

## Core Components

### GlassButton

Premium glassmorphism button with multiple variants and interactive states.

<Card title="Import">
  ```tsx import {GlassButton} from "@liquidify/components"; ```
</Card>

#### Props

| Prop           | Type                                                                          | Default     | Description            |
| -------------- | ----------------------------------------------------------------------------- | ----------- | ---------------------- |
| `variant`      | `'default' \| 'primary' \| 'secondary' \| 'destructive' \| 'ghost' \| 'link'` | `'default'` | Button style variant   |
| `size`         | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`                                        | `'md'`      | Button size            |
| `intensity`    | `'low' \| 'medium' \| 'high' \| 'ultra'`                                      | `'medium'`  | Glass effect intensity |
| `loading`      | `boolean`                                                                     | `false`     | Show loading state     |
| `disabled`     | `boolean`                                                                     | `false`     | Disable button         |
| `fullWidth`    | `boolean`                                                                     | `false`     | Full width button      |
| `icon`         | `ReactNode`                                                                   | -           | Icon element           |
| `iconPosition` | `'left' \| 'right'`                                                           | `'left'`    | Icon position          |

### GlassCard

Flexible card component with compound architecture.

<Card title="Import">
  ```tsx import {GlassCard} from "@liquidify/components"; ```
</Card>

#### Props

| Prop          | Type                                               | Default      | Description                |
| ------------- | -------------------------------------------------- | ------------ | -------------------------- |
| `variant`     | `'default' \| 'primary' \| 'secondary' \| 'apple'` | `'default'`  | Card style variant         |
| `size`        | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'`             | `'md'`       | Card size                  |
| `elevation`   | `'none' \| 'sm' \| 'md' \| 'lg' \| 'xl'`           | `'md'`       | Shadow elevation           |
| `interactive` | `boolean`                                          | `false`      | Enable hover/click effects |
| `selectable`  | `boolean`                                          | `false`      | Enable selection state     |
| `selected`    | `boolean`                                          | `false`      | Selection state            |
| `orientation` | `'vertical' \| 'horizontal'`                       | `'vertical'` | Card layout                |

#### Compound Components

- `GlassCard.Header` - Card header section
- `GlassCard.Title` - Card title
- `GlassCard.Description` - Card description
- `GlassCard.Content` - Main content area
- `GlassCard.Footer` - Footer section
- `GlassCard.Actions` - Action buttons area

## Event Handlers

Standard event handler patterns used across components:

```tsx
// Value change handlers
onValueChange?: (value: T) => void;
onCheckedChange?: (checked: boolean) => void;
onSelectedChange?: (selected: boolean) => void;

// State change handlers
onOpenChange?: (open: boolean) => void;
onExpandedChange?: (expanded: boolean) => void;
onActiveChange?: (active: boolean) => void;

// User interaction handlers
onClick?: (event: MouseEvent) => void;
onFocus?: (event: FocusEvent) => void;
onBlur?: (event: FocusEvent) => void;
onKeyDown?: (event: KeyboardEvent) => void;
```
