## 🎨 Customization Examples

<AccordionGroup>
  <Accordion title="Theme Customization" icon="palette">
    Learn how to customize the liquid glass theme system to match your brand.
    
    ```tsx
    import { GlassUIProvider } from '@liquidify/react';
    
    const customTheme = {
      colors: {
        primary: '#007AFF',
        surface: 'rgba(255, 255, 255, 0.1)',
        glassTint: 'rgba(0, 122, 255, 0.1)'
      },
      glassEffect: {
        opacity: 0.8,
        blur: '20px',
        saturation: 1.2
      }
    };
    
    function App() {
      return (
        <GlassUIProvider theme={customTheme}>
          {/* Your app content */}
        </GlassUIProvider>
      );
    }
    ```
  </Accordion>

  <Accordion title="Animation Configurations" icon="play">
    Configure spring physics and animation timing for your components.
    
    ```tsx
    import { GlassButton } from '@liquidify/react';
    
    function CustomAnimations() {
      return (
        <GlassButton
          animationConfig={{
            type: 'spring',
            stiffness: 300,
            damping: 20,
            mass: 1
          }}
          hoverScale={1.05}
          tapScale={0.95}
        >
          Custom Animation
        </GlassButton>
      );
    }
    ```
  </Accordion>

  <Accordion title="Accessibility Overrides" icon="universal-access">
    Override default accessibility settings for specific use cases.
    
    ```tsx
    import { GlassModal } from '@liquidify/react';
    
    function AccessibleModal() {
      return (
        <GlassModal
          aria-labelledby="modal-title"
          aria-describedby="modal-description"
          role="dialog"
          trapFocus={true}
          closeOnEscape={true}
          closeOnOutsideClick={false}
        >
          {/* Modal content with proper heading structure */}
        </GlassModal>
      );
    }
    ```
  </Accordion>
</AccordionGroup>
