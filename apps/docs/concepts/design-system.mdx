# Design System

LiqUIdify's design system combines Apple's Human Interface Guidelines with a unique liquid glass aesthetic to create modern, accessible, and engaging user interfaces.

## Core Principles

### 1. Clarity

- Clear visual hierarchy
- Purposeful use of space
- Consistent interaction patterns
- Readable typography
- High contrast when needed

### 2. Liquid Glass Aesthetic

- Multi-layered translucency
- Dynamic blur effects
- Smooth interactions
- Light refraction
- Environmental adaptation

### 3. Motion and Physicality

- Spring animations
- Liquid dynamics
- Natural motion
- Haptic feedback
- Responsive interactions

## Animation System

### Spring Animations

```tsx
const springConfigs = {
  gentle: {
    tension: 170,
    friction: 26,
  },
  bouncy: {
    tension: 300,
    friction: 10,
  },
  subtle: {
    tension: 120,
    friction: 14,
  },
};

function useSpringAnimation(config = "gentle") {
  return useSpring({
    config: springConfigs[config],
    from: { opacity: 0, scale: 0.9 },
    to: { opacity: 1, scale: 1 },
  });
}
```

## Best Practices

### Glass Effects

1. **Layering**
   - Use multiple glass layers for depth
   - Keep backdrop-filter performant
   - Consider z-index carefully

2. **Contrast**
   - Maintain readability
   - Test in different lighting
   - Support dark mode

3. **Performance**
   - Optimize animations
   - Limit blur radius
   - Use hardware acceleration

### Interactions

1. **Feedback**
   - Immediate visual response
   - Natural motion
   - Haptic feedback when available

2. **Accessibility**
   - High contrast when needed
   - Clear focus states
   - Motion reduction support

3. **Consistency**
   - Predictable behavior
   - Unified animation style
   - Coherent visual language

## Implementation Guide

### Setup

1. **Install Dependencies**
```bash
bun add @liquidify/components @liquidify/themes
```

2. **Theme Configuration**
```tsx
import { ThemeProvider, createTheme } from "@liquidify/components";

const theme = createTheme({
  glass: {
    blur: "12px",
    saturation: 1.8,
    brightness: 1.2,
  },
  // Other theme options...
});

function App() {
  return (
    \u003cThemeProvider theme={theme}\u003e
      \u003cYourApp /\u003e
    \u003c/ThemeProvider\u003e
  );
}
```
