## üé® Setup Provider

Wrap your application with the `UnifiedGlassProvider` to enable theme management and liquid glass effects:

<CodeGroup>

```tsx React
import { UnifiedGlassProvider } from "@liquidify/components";
import "@liquidify/components/css";

function App() {
  return <UnifiedGlassProvider>{/* Your app content */}</UnifiedGlassProvider>;
}
```

```tsx Next.js App Router
// app/layout.tsx
import { UnifiedGlassProvider } from "@liquidify/components";
import "@liquidify/components/css";

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        <UnifiedGlassProvider>{children}</UnifiedGlassProvider>
      </body>
    </html>
  );
}
```

```tsx Next.js Pages Router
// pages/_app.tsx
import { UnifiedGlassProvider } from "@liquidify/components";
import "@liquidify/components/css";

export default function App({ Component, pageProps }) {
  return (
    <UnifiedGlassProvider>
      <Component {...pageProps} />
    </UnifiedGlassProvider>
  );
}
```

</CodeGroup>

## üåô Apple-Style Dark Mode

LiqUIdify automatically adapts to your preferred color scheme with Apple's design principles:

```tsx
import { UnifiedGlassProvider, useTheme, GlassButton } from "@liquidify/components";

function ThemeToggle() {
  const { theme, setTheme } = useTheme();
  
  return (
    <GlassButton 
      onClick={() => setTheme(theme === 'light' ? 'dark' : 'light')}
      variant="ghost"
    >
      {theme === 'light' ? 'üåô' : '‚òÄÔ∏è'} Toggle Theme
    </GlassButton>
  );
}

function App() {
  return (
    <UnifiedGlassProvider>
      <div className="min-h-screen">
        <ThemeToggle />
        {/* Your app content */}
      </div>
    </UnifiedGlassProvider>
  );
}
```

## üíª TypeScript Support

LiqUIdify is built with TypeScript and provides excellent IntelliSense support:

```tsx
import { GlassButton, type GlassButtonProps } from "@liquidify/components";

// Full type safety with IntelliSense
const ButtonExample: React.FC<GlassButtonProps> = (props) => {
  return (
    <GlassButton
      variant="primary" // ‚úÖ Autocomplete: primary | secondary | ghost | destructive
      size="lg" // ‚úÖ Autocomplete: xs | sm | md | lg | xl
      disabled={false} // ‚úÖ Type-checked boolean
      glassEffect={{
        blur: 40, // ‚úÖ Type-checked number
        opacity: 0.1, // ‚úÖ Type-checked number
        borderOpacity: 0.2 // ‚úÖ Type-checked number
      }}
      {...props}
    >
      Click me
    </GlassButton>
  );
};
```

## ‚ôø Accessibility Built-In

LiqUIdify components follow WCAG 2.1 AA guidelines with Apple's accessibility standards:

```tsx
import { GlassButton, GlassModal } from "@liquidify/components";

function AccessibleModal() {
  const [isOpen, setIsOpen] = useState(false);
  
  return (
    <>
      <GlassButton 
        onClick={() => setIsOpen(true)}
        aria-describedby="modal-description"
      >
        Open Modal
      </GlassButton>
      
      <GlassModal
        isOpen={isOpen}
        onClose={() => setIsOpen(false)}
        aria-labelledby="modal-title"
        aria-describedby="modal-description"
      >
        <h2 id="modal-title">Accessible Modal</h2>
        <p id="modal-description">
          This modal includes proper ARIA attributes, focus management, 
          and respects prefers-reduced-motion settings.
        </p>
      </GlassModal>
    </>
  );
}
```

## üí° Pro Tips

<Tip>
**Performance**: Liquid glass effects use GPU acceleration for smooth 60fps animations. For best performance, limit the number of glass elements visible at once.
</Tip>

<Warning>
**Browser Support**: Backdrop-filter requires modern browsers. LiqUIdify gracefully degrades on older browsers with fallback styles.
</Warning>

<Note>
**Accessibility**: All glass effects maintain proper contrast ratios and work with screen readers. Use `prefers-reduced-motion` to respect user preferences.
</Note>
