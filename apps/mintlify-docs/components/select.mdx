## Props

| Prop               | Type                                  | Default              | Description                            |
| ------------------ | ------------------------------------- | -------------------- | -------------------------------------- |
| `options`          | `GlassSelectOption[]`                 | `[]`                 | Array of selectable options            |
| `value`            | `string \| string[]`                  | -                    | Current selected value(s)              |
| `onChange`         | `(value: string \| string[]) => void` | -                    | Callback when selection changes        |
| `placeholder`      | `string`                              | `'Select an option'` | Placeholder text                       |
| `disabled`         | `boolean`                             | `false`              | Disabled state                         |
| `multiple`         | `boolean`                             | `false`              | Enable multi-selection                 |
| `searchable`       | `boolean`                             | `false`              | Enable search functionality            |
| `maxSelections`    | `number`                              | -                    | Maximum selections (multi-select only) |
| `className`        | `string`                              | -                    | Additional CSS classes                 |
| `id`               | `string`                              | -                    | Input element ID                       |
| `aria-label`       | `string`                              | -                    | Accessibility label                    |
| `aria-describedby` | `string`                              | -                    | Accessibility description              |

### GlassSelectOption Interface

| Property   | Type      | Description                |
| ---------- | --------- | -------------------------- |
| `value`    | `string`  | Option value               |
| `label`    | `string`  | Option display text        |
| `disabled` | `boolean` | Whether option is disabled |

## Usage Patterns

### Form Integration

```tsx
function UserForm() {
  const [formData, setFormData] = useState({
    country: "",
    language: "",
    interests: [],
  });

  const countryOptions = [
    { value: "us", label: "United States" },
    { value: "ca", label: "Canada" },

  ];

  const languageOptions = [
    { value: "en", label: "English" },
    { value: "es", label: "Spanish" },
    { value: "fr", label: "French" },
  ];

  const interestOptions = [
    { value: "tech", label: "Technology" },
    { value: "design", label: "Design" },
```

### Dynamic Options

```tsx
import { useEffect, useState } from "react";
import { GlassFormField, GlassSelect } from "liquidify";

function DynamicSelect() {
  const [options, setOptions] = useState([]);
  const [loading, setLoading] = useState(true);
  const [value, setValue] = useState("");

  useEffect(() => {
    const fetchOptions = async () => {
      try {
        const response = await fetch("/api/options");
        const data = await response.json();
        setOptions(data);
      } catch (error) {
        console.error("Failed to fetch options:", error);
      } finally {
        setLoading(false);
      }
    };

    fetchOptions();
  }, []);

  if (loading) {
    return (
      <GlassSelect
        options={[{ value: "loading", label: "Loading...", disabled: true }]}
        disabled={true}
        placeholder="Loading options..."
      />
    );
  }

  return (
    <GlassSelect
      options={options}
      value={value}
      onChange={setValue}
      placeholder="Select an option"
      searchable={true}
    />
  );
}
```

### Controlled with Validation

```tsx
function ValidatedSelect() {
  const [value, setValue] = useState("");
  const [error, setError] = useState("");

  const options = [
    { value: "option1", label: "Option 1" },
    { value: "option2", label: "Option 2" },
    { value: "option3", label: "Option 3" },
  ];

  const handleChange = (newValue) => {
    setValue(newValue);

    // Validation
    if (!newValue) {
      setError("Please select an option");
    } else {
      setError("");
    }
  };

  return (
    <GlassFormField
      label="Required Selection"
      required
      error={error}
      success={value && !error ? "Valid selection" : undefined}
    >
      <GlassSelect
        options={options}
        value={value}
        onChange={handleChange}
        placeholder="Select an option"
      />
    </GlassFormField>
  );
}
```

## Accessibility

### WCAG Compliance

- **Keyboard Navigation**: Full keyboard support with arrow keys
- **Screen Reader**: Proper ARIA attributes and announcements
- **Focus Management**: Clear focus indicators and logical tab order
- **Selection Announcement**: Screen readers announce selection changes

### Keyboard Shortcuts

| Key           | Action                              |
| ------------- | ----------------------------------- |
| `Space/Enter` | Open/close dropdown                 |
| `Arrow Down`  | Navigate to next option             |
| `Arrow Up`    | Navigate to previous option         |
| `Enter`       | Select focused option               |
| `Escape`      | Close dropdown                      |
| `Home`        | Focus first option                  |
| `End`         | Focus last option                   |
| `Type`        | Search for option (when searchable) |

## Apple HIG Guidelines

### Visual Design

- **Clarity**: Clear visual hierarchy in dropdown options
- **Depth**: Subtle glass effects that enhance the selection metaphor
- **Consistency**: Consistent styling across all states
- **Feedback**: Immediate visual feedback for selections

### Interaction Design

- **Predictable**: Familiar dropdown behavior
- **Efficient**: Quick selection with keyboard shortcuts
- **Forgiving**: Clear indication of disabled options
- **Responsive**: Smooth animations and transitions

### Content Organization

- **Logical Order**: Options ordered logically (alphabetical, by importance)
- **Clear Labels**: Descriptive option text
- **Grouping**: Related options grouped when appropriate
- **Search**: Search functionality for long lists

## Best Practices

### Do's

- ✅ Use clear, descriptive option labels
- ✅ Provide search for long option lists
- ✅ Handle loading and error states
- ✅ Use appropriate placeholder text
- ✅ Implement proper keyboard navigation

### Don'ts

- ❌ Don't use select for binary choices (use switch/checkbox)
- ❌ Don't make dropdown lists too long without search
- ❌ Don't forget to handle empty states
- ❌ Don't rely only on color for disabled states
- ❌ Don't ignore mobile touch targets

### Performance

- **Virtual Scrolling**: For very long option lists
- **Debounced Search**: Prevent excessive filtering
- **Memoized Options**: Cache processed options
- **Lazy Loading**: Load options on demand

```tsx
// Performance optimizations
const memoizedOptions = useMemo(
  () =>
    rawOptions.map((option) => ({
      ...option,
      searchText: option.label.toLowerCase(),
    })),
  [rawOptions],
);

const debouncedFilter = useDebouncedCallback((query) => {
  const filtered = memoizedOptions.filter((option) =>
    option.searchText.includes(query.toLowerCase()),
  );
  setFilteredOptions(filtered);
}, 200);
```
